## 1. グラフの行列表現

### 隣接行列 \(A = [a_{ij}]\)

- \(a_{ij}=1\): ノード i → j へ有向エッジあり
- 無向グラフ → **対称行列**（\(a_{ij}=a_{ji}\)）
- 有向グラフ → **非対称**（因果・物流）
- \(a_{ii}=1\): 自己ループ（ISM入力で使用）

### 3種類の行列

| 種類 | 値 | 用途 |
|------|-----|------|
| バイナリ | 0/1 | ISM：到達性・階層 |
| 重み付き | 実数（0〜4等） | DEMATEL：影響の強さ |
| 距離 | 最短距離 | クラスタリング |

### ブール代数演算

- **和**: \(A \lor B\)（OR）— どちらかのパスで到達  
- **積**: \(A \land B\)（AND）  
- 行列べき乗 \(A^k\) の \((i,j)\) 成分 = **kステップ以内**に i から j へ到達可能

## 2. 可到達行列 \(M_R\)

\[
M = A + I \quad (\text{単位行列を加え自己到達を含む})
\]

\(M, M^2, M^3, \ldots\) をブール和で計算し、**変化がなくなるまで**繰り返す → 可到達行列 \(M_R\)。

- 推移律が成り立つ n ノードグラフでは **n−1 回**程度で収束
- ISMでは一対比較から得た構造行列に適用

### 収束の見方

\(M^{(k)} = M^{(k-1)}\) なら収束。小規模例では手計算で確認。

## 3. 階層分割（ISMの核心）

各要素 \(s_i\) について:

| 記号 | 定義 |
|------|------|
| **R(i)** | i から到達可能な要素の集合 |
| **A(i)** | i に到達可能な要素の集合（iより「上」） |
| **R∩A** | 両者の共通部分 |

**抽出条件**: \(R(s_i) \cap A(s_i) = R(s_i)\) を満たす要素を**同一レベル**として取り除く。残りで繰り返す。

### 講義例（6要素 S1–S6）

| Level | 要素 | 解釈 |
|-------|------|------|
| ① | S5 | 最上位（結果側） |
| ② | S1, S3, S4 | |
| ③ | S2 | |
| ④ | S6 | 最下位（原因側） |

## 4. ブロック三角化と骨格行列

- **ブロック三角化**: 可到達行列の行・列を並べ替え、強連結成分（ループを持つ塊）をまとめる
- **骨格行列（スケルトン行列）**: 冗長な推移関係を除いた**最小限**の直接関係のみを残す階層図の骨格

---

## 5. ISM法のプロセス

1. 要素抽出  
2. 一対比較（人間）→ 構造行列（0/1）  
3. 可到達行列の計算（コンピュータ）  
4. 多階層分割（R∩A=R）  
5. 多層有向グラフ・骨格行列  
6. **メンタルモデルとの比較・修正**（重要）

### 推移律

\(s_i R s_j\) かつ \(s_j R s_k\) なら \(s_i R s_k\)。  
→ 一対比較は理論上 \(n(n-1)\) 回だが、**1/2〜1/3に削減**可能。

| n | 理論値 | 実際（目安） |
|---|--------|--------------|
| 4 | 12 | 4〜6 |
| 8 | 56 | 19〜28 |
| 14 | 182 | 61〜91 |

### ISMが使えない例

- SNSのフォロー（推移律なし）→ **DEMATEL**
- 「友人」「似ている」など

## 6. DEMATEL法

### 手順（試験必須）

| Step | 内容 |
|------|------|
| 1 | 要素抽出 |
| 2 | 専門家が i→j の影響を **0〜4** で評価 |
| 3 | 直接影響行列 X → **正規化** N（行最大和で割る） |
| 4 | **T = N(I−N)⁻¹** |
| 5 | D=行和, R=列和 |
| 6 | **影響度**=D−R, **関連度**=D+R → 散布図 |

### ISM との比較

| | ISM | DEMATEL |
|---|-----|---------|
| 関係 | 0/1 | 0〜4 |
| 推移律 | 仮定 | 仮定しない |
| 入力 | 対話的一対比較 | アンケート |
| 目的 | **階層化** | **影響度・原因/結果の定量** |

### 正規化の意味

行の最大和で割る → N のべき乗が収束（スペクトル半径 < 1）→ \((I-N)^{-1}\) が存在。

## 7. 使い分けフローチャート

```
推移律は成り立つか？
  No → DEMATEL
  Yes → 階層構造が必要か？
    Yes → ISM
    No → 影響の強さ・原因系/結果系が必要か？
      Yes → DEMATEL
      No → 因果の正負が重要か？
        Yes → Cognitive Map
```

**併用**: ISMで階層 + CMで符号 → 設計トレードオフの把握（ノートPC例）。
