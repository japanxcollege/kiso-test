# 試験情報と対策方針

## 成績構成

| 項目 | 配点 | 備考 |
|------|------|------|
| 出席・レポート | 60点 | 青山 25点相当（5/12）＋ 藤井 35点相当（7/12） |
| 期末試験 | 40点 | **講義全体**から出題・持ち込みなし |
| 合計 | 100点 | A/B/C 評価 |

## 出題の想定

### 青山担当（構造分析・設計）

- システムの定義、境界、システムズアプローチ
- 隣接行列・可到達行列・階層分割（R∩A）
- **ISM法**の手順・推移律・使い分け
- **DEMATEL法**の計算（N, T, D, R, D±R）と原因系/結果系
- Cognitive Map、因果ループ（R/B）、SD（ストック・フロー）
- ISM / DEMATEL / CM の使い分け

### 藤井担当（シミュレーション）

- 確率モデル・確率過程の定義
- マルコフ性、推移確率行列、定常分布
- 待ち行列の直観（利用率・待ち時間）
- オートマトン・セルオートマトン（ルール・Neumann近傍）
- エージェント・MAS、自己組織化臨界・べき乗則
- 連続体の離散化（格子・有限差分の考え方）

## 優先度

| 優先 | 内容 |
|------|------|
| 最優先 | DEMATEL 計算手順、ISM の R∩A 階層化、マルコフ性の定義 |
| 高 | 可到達行列、因果ループ R/B、使い分けフローチャート |
| 中 | SD ストック/フロー、サンドパイル/SOC、待ち行列 |
| 補完 | ファジィ・AHP（資料薄い場合はシラバスレベル） |
