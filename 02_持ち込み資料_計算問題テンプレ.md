## 1. DEMATEL の全計算（最重要・最頻出）

### 標準問題

```
3要素 ①②③ の直接影響行列が次のように与えられた：
X = [[0, x12, x13],
     [x21, 0, x23],
     [x31, x32, 0]]
（各要素は 0〜4 の値）

(問1) 正規化行列 N を求めよ
(問2) 総合影響行列 T = N(I−N)⁻¹ を求めよ
(問3) D, R を求めよ
(問4) D−R と D+R を計算し、原因系/結果系を判別せよ
```

### 完全解答テンプレ（基本問題 A-4 の例）

**問題**：X = [[0, 2, 0], [0, 0, 3], [0, 0, 0]]

---

#### 問1：正規化行列 N

**ステップ 1**：各行の行和を計算する。

- 1 行目：0 + 2 + 0 = **2**
- 2 行目：0 + 0 + 3 = **3**
- 3 行目：0 + 0 + 0 = **0**

**ステップ 2**：行和の最大値を求める。

最大行和 = **3**

**ステップ 3**：X を 3 で割って正規化する。

$$
N = \frac{X}{3} = \begin{pmatrix} 0 & 2/3 & 0 \\ 0 & 0 & 1 \\ 0 & 0 & 0 \end{pmatrix}
$$

**なぜ正規化するか**：次のステップで N の冪和を取るが、N の固有値が 1 未満であることを保証するため。

---

#### 問2：総合影響行列 T

**ステップ 1**：N² を計算する。

$$
N^2 = \begin{pmatrix} 0 & 2/3 & 0 \\ 0 & 0 & 1 \\ 0 & 0 & 0 \end{pmatrix}^2 = \begin{pmatrix} 0 & 0 & 2/3 \\ 0 & 0 & 0 \\ 0 & 0 & 0 \end{pmatrix}
$$

（計算例：(1,3) 成分 = 0×0 + 2/3 × 1 + 0×0 = 2/3）

**ステップ 2**：N³ を計算する。

$$
N^3 = N \cdot N^2 = 0
$$

N が **冪零**（N³ = 0）。

**ステップ 3**：(I−N)⁻¹ を計算する。

N が冪零なので、

$$
(I - N)^{-1} = I + N + N^2 = \begin{pmatrix} 1 & 2/3 & 2/3 \\ 0 & 1 & 1 \\ 0 & 0 & 1 \end{pmatrix}
$$

**ステップ 4**：T を計算する。

$$
T = N(I - N)^{-1} = N + N^2 + N^3 + \ldots = N + N^2 = \begin{pmatrix} 0 & 2/3 & 2/3 \\ 0 & 0 & 1 \\ 0 & 0 & 0 \end{pmatrix}
$$

---

#### 問3：D と R

**D（行和）**：

- D₁ = 0 + 2/3 + 2/3 = **4/3**
- D₂ = 0 + 0 + 1 = **1**
- D₃ = 0 + 0 + 0 = **0**

**R（列和）**：

- R₁ = 0 + 0 + 0 = **0**
- R₂ = 2/3 + 0 + 0 = **2/3**
- R₃ = 2/3 + 1 + 0 = **5/3**

---

#### 問4：D−R と D+R、原因系/結果系の判別

| 要素 | D | R | **D+R** | **D−R** | 判定 |
|------|---|---|---------|---------|------|
| ① | 4/3 | 0 | **4/3** | **+4/3** | 最も**原因系**（発信のみ） |
| ② | 1 | 2/3 | **5/3** | **+1/3** | やや原因系（中継的） |
| ③ | 0 | 5/3 | **5/3** | **−5/3** | 最も**結果系**（受信のみ） |

**解釈**：
- ① が D−R の値が最も大きく、システムの**根本原因**となる要素。介入するならここ。
- ③ が D−R の値が最も小さく、**症状にあたる要素**。対症療法しか効かない。
- ② は中継的な位置。

---

### 計算で間違えやすいポイント

1. **行和の最大値で割る**（列和ではない）
2. **N の対角要素は通常 0**（自己への影響は問題で 0 と指定される）
3. **行列積の計算**：A·B の (i,j) 成分 = A の i 行と B の j 列の内積
4. **D は行和（発信）、R は列和（被影響）** — 混同しないように
5. **D−R の符号**：正が原因系、負が結果系

## 2. マルコフ定常分布の計算

### 標準問題

```
推移行列 P の定常分布 π = (π₁, π₂) を求めよ。
ただし πP = π かつ Σπ_i = 1。
```

### 完全解答テンプレ

**問題**：P = [[0.9, 0.1], [0.2, 0.8]]

---

**ステップ 1**：推移行列の確認（各行の和 = 1）

- 1 行目：0.9 + 0.1 = 1 ✓
- 2 行目：0.2 + 0.8 = 1 ✓

**ステップ 2**：πP = π を成分ごとに書く

π = (π₁, π₂) として、

$$
\begin{aligned}
\pi_1 \cdot 0.9 + \pi_2 \cdot 0.2 &= \pi_1 \\
\pi_1 \cdot 0.1 + \pi_2 \cdot 0.8 &= \pi_2
\end{aligned}
$$

**ステップ 3**：1 つの式を整理する（どちらか一方でよい）

第 1 式から：

$$
0.9\pi_1 + 0.2\pi_2 = \pi_1 \quad \Rightarrow \quad 0.2\pi_2 = 0.1\pi_1 \quad \Rightarrow \quad \pi_1 = 2\pi_2
$$

**ステップ 4**：正規化条件 π₁ + π₂ = 1 を使う

$$
2\pi_2 + \pi_2 = 1 \quad \Rightarrow \quad 3\pi_2 = 1 \quad \Rightarrow \quad \pi_2 = \frac{1}{3}
$$

したがって π₁ = 2/3。

**答え**：

$$
\boxed{\pi = \left( \frac{2}{3}, \frac{1}{3} \right)}
$$

---

### 別の数値例（練習用）

**問題**：P = [[0.7, 0.3], [0.4, 0.6]]

- 0.7π₁ + 0.4π₂ = π₁ → 0.4π₂ = 0.3π₁ → π₁ = (4/3)π₂
- π₁ + π₂ = 1 → (4/3)π₂ + π₂ = 1 → (7/3)π₂ = 1 → π₂ = 3/7
- π₁ = 4/7

**答え**：π = (4/7, 3/7)

---

### 3×3 の場合（出題されたとき用）

**問題**：P = [[0.5, 0.3, 0.2], [0.1, 0.6, 0.3], [0.2, 0.4, 0.4]]

π = (π₁, π₂, π₃)

連立：
- 0.5π₁ + 0.1π₂ + 0.2π₃ = π₁
- 0.3π₁ + 0.6π₂ + 0.4π₃ = π₂
- 0.2π₁ + 0.3π₂ + 0.4π₃ = π₃
- π₁ + π₂ + π₃ = 1

→ 上の3つから2つを選び、正規化条件と合わせて3元連立を解く。

整理すると：
- −0.5π₁ + 0.1π₂ + 0.2π₃ = 0
- 0.3π₁ − 0.4π₂ + 0.4π₃ = 0
- π₁ + π₂ + π₃ = 1

→ 計算機を使ってもよい（持ち込み PC で Excel 等）。

---

### よくある間違い

1. **πP と Pπ を混同**：定常分布は **πP = π**（行ベクトルとして右から P を掛ける）
2. **各行の和 = 1** の確認を忘れる
3. **正規化条件 Σπ_i = 1** を忘れる
4. **連立方程式が冗長**：πP = π は実質 n − 1 個の独立な式

## 3. M/M/1 待ち行列

### 標準問題

```
M/M/1 待ち行列で、到着率 λ = 8（人/時）、サービス率 μ = 10（人/時）。
(1) 利用率 ρ を求めよ
(2) 系が安定かどうか判定せよ
(3) リトルの法則を用いて、平均滞在時間 W を求めよ
```

### 完全解答テンプレ

**(1) 利用率 ρ**

$$
\rho = \frac{\lambda}{\mu} = \frac{8}{10} = \boxed{0.8}
$$

**(2) 安定性の判定**

ρ = 0.8 < 1 なので、系は**安定**である。

（ρ ≥ 1 だと到着が処理を上回り、待ち行列が無限に伸びて発散する。）

**(3) 平均滞在時間 W**

M/M/1 の公式より、

$$
L = \frac{\rho}{1 - \rho} = \frac{0.8}{0.2} = 4 \text{（人）}
$$

リトルの法則 L = λW より、

$$
W = \frac{L}{\lambda} = \frac{4}{8} = 0.5 \text{ 時間} = 30 \text{ 分}
$$

または直接公式：

$$
W = \frac{1}{\mu - \lambda} = \frac{1}{10 - 8} = 0.5 \text{ 時間}
$$

## 4. 可到達行列の計算

### 標準問題

```
4 ノードの有向グラフの隣接行列 A が与えられた。
可到達行列を求めよ。
```

### 完全解答テンプレ

**手順**：

1. M = A + I を計算する
2. M² を計算する（M × M）
3. M³ を計算する
4. M^k = M^(k+1) になった時点で収束
5. 4 ノードなら n − 1 = 3 回程度で収束

**注意**：行列の積の各成分を計算するときは、

$$
(M^2)_{ij} = \sum_k M_{ik} M_{kj}
$$

ブール演算（0 か 1 か）で計算する場合は、和を OR、積を AND として、最終結果は 0/1 にする。

## 5. ISM の階層抽出

### 標準問題

```
要素 a, b, c, d の可到達行列が与えられた。
各要素のレベルを求めよ。
```

### 完全解答テンプレ

**手順**：

1. 各要素 i について R(i) と A(i) を可到達行列から読み取る：
   - **R(i)**：i の行で「1」がついている要素（i 自身も含む）
   - **A(i)**：i の列で「1」がついている要素（i 自身も含む）
2. **R(i) ∩ A(i) = R(i)** となる要素を抽出（最上位レベル）
3. 抽出した要素を除いて残りの要素で同じ判定を繰り返す
4. すべての要素が階層化されるまで反復

**例**（架空の数値）：

要素 4 つで、

| | R(i) | A(i) | R ∩ A | R∩A=R? |
|---|------|------|-------|--------|
| a | {a, b, c, d} | {a} | {a} | × |
| b | {b, c, d} | {a, b} | {b} | × |
| c | {c} | {a, b, c, d} | {c} | ✓ → **Level 1**（最上位） |
| d | {d} | {a, b, d} | {d} | ✓ → **Level 1** |

→ Level 1 = {c, d}。これを除外して、a, b で再判定。

| | R(i) | A(i) | R ∩ A | R∩A=R? |
|---|------|------|-------|--------|
| a | {a, b} | {a} | {a} | × |
| b | {b} | {a, b} | {b} | ✓ → **Level 2** |

→ Level 2 = {b}。残る a が Level 3。

階層図：

```
Level 1: c, d ← 最上位（最も多くの要素から影響を受ける）
Level 2: b
Level 3: a ← 最下位（最も多くの要素に影響を与える）
```

**注意**：「最上位」の意味は文脈による。ISM では「**下流の終着点**＝結果系」を最上位とすることが多い。

## 6. 因果ループの R/B 判定

### 標準問題

```
次の連鎖が強化ループ(R)かバランスループ(B)か判定せよ。
「A →(O) B →(S) C →(S) A」
```

### 完全解答テンプレ

**ステップ 1**：S と O を数える

- リンク 1：A →(O) B（O）
- リンク 2：B →(S) C（S）
- リンク 3：C →(S) A（S）

O の個数 = **1 個（奇数）**

**ステップ 2**：判定ルールを適用

- O 偶数個 → R（強化ループ）
- O 奇数個 → B（バランスループ）

→ **B（バランスループ）**

**ステップ 3**：解釈

A が増えると、B が減る（O）→ B が減ると C も減る（S）→ C が減ると A も減る（S）。つまり A の増加が**自分自身の減少**を引き起こす **負のフィードバック**で、A は均衡値に収束する。

**もう一つの判定法**：S = +1、O = −1 として全体の積を取る。

積 = (−1)(+1)(+1) = **−1** → **B**

---

### 別の例

「ストレス↑ →(O) 睡眠↓ →(O) 疲労↑ →(S) ストレス↑」

- O の個数 = 2 個（偶数） → **R（強化ループ）**
- 積 = (−1)(−1)(+1) = +1 → **R**

ストレスが自己強化する悪循環。

## 7. AHP の重み計算（簡略法）

### 標準問題

```
3 つの基準について一対比較行列が与えられた。
各基準の重みを求めよ。
```

### 完全解答テンプレ（簡略幾何平均法）

**問題**：

$$
A = \begin{pmatrix} 1 & 3 & 5 \\ 1/3 & 1 & 3 \\ 1/5 & 1/3 & 1 \end{pmatrix}
$$

**ステップ 1**：各行の幾何平均を計算

- 行 1：(1 × 3 × 5)^(1/3) = 15^(1/3) ≈ **2.466**
- 行 2：(1/3 × 1 × 3)^(1/3) = 1^(1/3) = **1.000**
- 行 3：(1/5 × 1/3 × 1)^(1/3) = (1/15)^(1/3) ≈ **0.405**

**ステップ 2**：合計を計算

2.466 + 1.000 + 0.405 ≈ **3.871**

**ステップ 3**：正規化（合計で割る）

- w₁ = 2.466 / 3.871 ≈ **0.637**
- w₂ = 1.000 / 3.871 ≈ **0.258**
- w₃ = 0.405 / 3.871 ≈ **0.105**

→ 重みベクトル w = (0.637, 0.258, 0.105)

**ステップ 4**：CR の計算（必要なら）

CR < 0.1 なら判断は一貫している。0.1 を超えたら判断を見直す。

## 8. サンドパイル・モデルの更新

### 標準問題

```
2 次元の格子（各セルの値が砂の高さ）で、サンドパイルの更新規則を述べよ。
```

### 完全解答テンプレ

**更新規則の 3 ステップ**：

1. **砂粒を 1 つ落とす**：ランダムに 1 セルを選び、その値を +1 する
2. **崩壊条件のチェック**：値が **4 以上**になったセルがあれば、そのセルから 4 を減らし、上下左右の 4 近傍（von Neumann 近傍）に +1 ずつ加える
3. **連鎖崩壊の処理**：崩壊によって他のセルも 4 以上になれば、すべて 3 以下に戻るまで 2 を繰り返す

安定したら次の砂粒を落とし、1 に戻る。

**結果**：

- 砂山は自然に**一定の傾き**を保つ（**臨界状態**）
- 崩壊の規模が**べき乗分布**になる：小さな崩壊も大規模な雪崩も同じ法則
- **自己組織化臨界（SOC）** の典型例

## 9. その他の典型計算

### l ステップ推移

P = [[0.9, 0.1], [0.2, 0.8]]

「2 ステップで状態 1 から状態 2 に行く確率」：

$$
p_{12}^{(2)} = (P^2)_{12}
$$

P² を計算：

P² の (1,2) 成分 = 0.9 × 0.1 + 0.1 × 0.8 = 0.09 + 0.08 = **0.17**

### 期待値（離散）

確率質量関数 p(x) が与えられたとき：

$$
E[X] = \sum_i x_i p(x_i)
$$

例：サイコロの目の期待値 = 1×(1/6) + 2×(1/6) + ... + 6×(1/6) = **3.5**

### 二項分布の期待値・分散

X ~ B(n, p) なら、

- E[X] = np
- V[X] = np(1−p)

例：n=10, p=0.3 → E=3, V=2.1

## 試験中の注意

1. **問題文を最初によく読む**：与えられた行列、変数を確認
2. **途中式を必ず残す**：部分点が来る
3. **計算結果は分数で書く**（小数より厳密）
4. **検算する**：N の行和、π の和 = 1 など
5. **時間切れに注意**：計算 1 問に 20 分以上かけない
