### T−12h 〜 T−9h（3h）：A の計算問題を完璧にする

#### ① 試験総合対策.md の全体像を 10 分だけ読む（10分）

#### ② DEMATEL の計算を**5回**繰り返す（90分）

これだけで試験の 1/4 が取れる可能性がある。

**手順**：

1. X（直接影響行列、0〜4 で評価された生データ）
2. 各行の行和 → 最大行和で全体を割る = **N**
3. (I−N)⁻¹ を計算（N が冪零なら **I + N + N²** で済む）
4. **T = N(I−N)⁻¹**
5. D = 行和、R = 列和
6. **D + R（関連度）、D − R（因果性）**
7. **D−R > 0 が原因系、D−R < 0 が結果系**

**コツ**：行列を書くときは、**N の対角要素は必ず 0**（自分への影響は通常 0）。N² は素直に行列の積。

**5回繰り返す問題**：

- 1回目：基本問題 A-4（X = [[0,2,0],[0,0,3],[0,0,0]]）
- 2回目：少し変えた問題（自分で作る）
- 3回目：もう一度同じ問題を見ずに
- 4回目：用語・公式一覧.md の練習問題
- 5回目：時間制限 10 分で

5回目で **10分以内に正解できる**ようになれば合格。

#### ③ マルコフ定常分布の計算を **3回**繰り返す（45分）

**手順**：

1. 推移行列 P（各行の和 = 1 を確認）
2. πP = π を成分ごとに書く
3. Σπ_i = 1 を加える
4. 連立方程式を解く

**3つの問題**：

- P = [[0.9, 0.1], [0.2, 0.8]] → π = (2/3, 1/3)
- P = [[0.7, 0.3], [0.4, 0.6]] → π = (4/7, 3/7)
- 自分で 3×3 を作って解く

時間制限 5 分で解けるように。

### T−9h 〜 T−6h（3h）：論述系の「絶対書ける」テンプレを身につける

#### ⑤ 用語説明の「テンプレ」を3つ覚える（45分）

試験総合対策.md から3つの用語を選んで、**1分以内で4文書ける**ようにする。

**用語1：システム3要素**

> 「**システム**とは、要素・相互作用・目的性の3つを持つ全体である（定義）。要素だけではシステムとは呼ばず、要素間の相互作用と全体の目的があって初めてシステムになる（重要性）。例えば自動車システムなら、エンジンやタイヤが要素、力の伝達が相互作用、人を運ぶのが目的（具体例）。境界の引き方は分析の出発点であり、観点で変わる（関連概念）。」

**用語2：マルコフ性**

> 「**マルコフ性**とは、確率過程において次の状態が現在の状態だけで決まり、過去の履歴に依存しない性質（定義）。これにより過去の全履歴を覚えなくても遷移確率だけで未来を予測できる（重要性）。例えば天気の遷移で、今日が晴れなら明日の確率分布が決まり、昨日や一昨日の情報は不要（具体例）。これが推移行列 P と定常分布 πP=π の解析を可能にする（関連概念）。」

**用語3：DEMATEL**

> 「**DEMATEL**とは、要素間の影響を 0〜4 で評価し、直接影響と間接影響の総和を計算する手法（定義）。ISM が階層化を目的とするのに対し、DEMATEL は影響の強さの定量と原因系/結果系の判別を目的とする（重要性）。直接影響行列 X を正規化して N を求め、T = N(I−N)⁻¹ で総合影響を計算し、行和 D と列和 R から D−R で原因/結果を判別する（具体例）。Battelle Geneva が 1972 年に開発した（関連概念）。」

この3つを**自分の言葉で言える**まで反復。試験で似た用語が出たら、この型に当てはめて書く。

#### ⑥ 「誤りを指摘」型の典型パターンを2つ覚える（30分）

**誤り1：「DEMATEL は因果の正負を + と − で表す」**

> 誤り。DEMATEL は**非負の影響の強さ（0〜4）**を扱う。正負の符号を扱うのは **Cognitive Map（CM）** や **FCM**。

**誤り2：「マルコフ連鎖では過去の全履歴が次の状態を決める」**

> 誤り。マルコフ性により、**現在の状態のみ**が次の状態を決定する。過去の履歴（X_{n-1}, X_{n-2}, ...）は不要。式で書けば P(X_{n+1}=j | X_n=i, X_{n-1}, ...) = P(X_{n+1}=j | X_n=i)。

この2つは確実に出題される可能性が高い。**3 〜 4 点**は確保できる。

#### ⑦ 対比の表を3つ覚える（30分）

論述問題で「○○と××の違いを述べよ」と問われたら、表で書くのが一番速くて点が来る。

**ISM vs DEMATEL**：

| | ISM | DEMATEL |
|---|---|---|
| 関係の表現 | 0/1（あり/なし） | 0〜4（強さ） |
| 推移律 | 仮定する | 仮定しない |
| 主目的 | 階層化 | 影響度・原因/結果の定量 |
| 出力 | 階層図 | D−R/D+R グラフ |

**確率 vs ファジィ**：

| | 確率 | ファジィ |
|---|---|---|
| 対象 | 事象の発生の不確実性 | 概念の曖昧さ・程度 |
| 例 | 雨が降る確率 60% | 「高齢」の度合い 0.7 |

**ストック vs フロー**：

| | ストック | フロー |
|---|---|---|
| 意味 | 時点で測れる蓄積量 | 単位時間あたりの増減 |
| 例 | 人口、貯金、CO₂濃度 | 出生率、入金、排出率 |
| 性質 | 過去の蓄積を持つ | ストックを変化させる |

この3つの対比は試験で**直接問われる可能性が高い**。

#### ⑧ S/O と R/B の判定ルールを覚える（20分）

- **S（Same）**：同方向変化（A 増 → B 増）
- **O（Opposite）**：逆方向変化（A 増 → B 減）
- **R（強化ループ）= O が偶数個**
- **B（バランスループ）= O が奇数個**

例題を3つ自分で解く：

1. 「貯金 →(S) 利息 →(S) 貯金」→ O が0個（偶数）→ **R**
2. 「体温 →(S) 発汗 →(O) 体温」→ O が1個（奇数）→ **B**
3. 「ストレス →(O) 睡眠 →(O) 疲労 →(S) ストレス」→ O が2個（偶数）→ **R**

## 一番のリスク

「**計算問題が想定と違う形式で出る**」がリスク。たとえば：

- DEMATEL が 4×4 で出題される → 計算量が増えて時間切れ
- 定常分布が 3×3 で出る → 連立方程式が複雑化

これに備えて、

- **DEMATEL は 3×3 を完璧にしたうえで、4×4 も1回やっておく**
- **定常分布は 2×2 と 3×3 の両方を解いておく**

これで計算問題のサプライズは大体カバーできる。
