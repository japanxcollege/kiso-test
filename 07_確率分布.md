## 1. 分布関数 ⭐

\[
F(x) = P(X \leq x)
\]

**累積分布関数（CDF）** — 離散・連続のどちらにも定義可能。

### 性質

- 広義単調増加
- 右連続
- \(\lim_{x\to-\infty} F(x)=0\), \(\lim_{x\to\infty} F(x)=1\)
- \(P(a < X \leq b) = F(b) - F(a)\)

---

## 2. 離散型確率変数

確率質量関数: \(p(x) = P(X=x)\)

\[
F(x) = \sum_{x_k \leq x} p(x_k)
\]

### 例: 二項分布 \(B(n, p)\)

n回の独立試行で成功回数。\(P(X=k) = \binom{n}{k} p^k (1-p)^{n-k}\)

---

## 3. 連続型確率変数

密度関数 \(f(x) \geq 0\), \(\int_{-\infty}^{\infty} f(x)\,dx = 1\)

\[
F(x) = \int_{-\infty}^{x} f(y)\,dy, \quad P(a \leq X \leq b) = \int_a^b f(x)\,dx
\]

**注意**: 連続型では \(P(X=x)=0\)。確率は**区間の積分**で与えられる。

---

## 4. 期待値・分散

| | 離散 | 連続 |
|---|------|------|
| 期待値 | \(E[X]=\sum x_i p(x_i)\) | \(E[X]=\int x f(x)\,dx\) |
| 分散 | \(V[X]=E[(X-E[X])^2]\) | 同左 |

**線形性**: \(E[aX+b] = aE[X]+b\)

---

## 5. 代表分布（試験で名前と用途）

| 分布 | パラメータ | 用途のイメージ |
|------|------------|----------------|
| ベルヌーイ | \(p\) | 1回の成功/失敗 |
| 二項 | \(n, p\) | n回独立試行の成功数 |
| ポアソン | \(\lambda\) | 単位時間の稀な事象数 |
| 幾何 | \(p\) | 初成功までの試行回数 |
| 指数 | \(\lambda\) | 待ち時間（**無記憶性**） |
| 正規 | \(\mu, \sigma^2\) | 測定誤差、中心極限定理 |

### 指数分布の無記憶性

\(P(X>s+t \mid X>s) = P(X>t)\) — マルコフ性と概念的に近い。

---

## 6. 数値計算・シミュレーション

### 乱数生成

コンピュータは**疑似乱数**で確率過程を実現。

### モンテカルロ法

- 乱数サンプルで期待値・確率を近似
- サンプル数を増やすと精度向上（大数の法則）

### 講義との関係

第8回マルコフ連鎖、第9回CA、第11回SOC — いずれも**乱数＋離散更新**でシミュレーション。
