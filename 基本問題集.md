# システム工学基礎 — 基本問題集

## A. 構造分析（青山担当）

### A-1 システムの定義

**問1** システムを構成する3つの要素を挙げ、それぞれ簡潔に説明せよ。

**問2** 存在論的な境界と認識論的な境界の違いを述べよ。

**問3** 構造モデリング（SM）と動的モデリング（DM）の違いを、焦点・代表的手法・目的の観点から述べよ。

**問4** 良定義問題と悪定義問題（wicked problem）の違いを1文ずつ述べよ。

### A-2 グラフと可到達行列

4ノードの有向グラフについて、隣接行列 \(A\) が与えられたとき、次に答えよ。

**問1** \(a_{ij}\) の意味を述べよ。

**問2** 可到達行列を求めるために \(M = A + I\) の何乗を計算するか。推移律が成り立つ n ノードの場合、何回程度で収束するか。

**問3** ブロック三角化と骨格行列を、それぞれ1文で説明せよ。

### A-3 ISM法

**問1** ISM法が前提とする「推移律」を定義せよ。推移律が成り立たない関係の例を1つ挙げよ。

**問2** 可到達行列から階層分割するとき、各レベルで抽出する条件 **R ∩ A = R** の意味を、R（到達可能集合）と A（先行集合）を用いて説明せよ。

**問3** ISM法と DEMATEL法を、関係の表現（0/1か多段階か）、推移律、主目的の4点で比較せよ。

### A-4 DEMATEL法（計算）

3要素（①②③）の直接影響行列 X を次とする（行→列の影響度 0〜4）:

$$
X = \begin{pmatrix}
0 & 2 & 0 \\
0 & 0 & 3 \\
0 & 0 & 0
\end{pmatrix}
$$

**問1** 正規化行列 N を求めよ（各行の最大行和で除算）。

**問2** 総合影響行列 \(T = N(I-N)^{-1}\) を求めよ。

**問3** D（行和）, R（列和）を求め、各要素の影響度 D−R と関連度 D+R を述べよ。どの要素が原因系・結果系か議論せよ。

### A-5 因果ループ・SD

**問1** 因果ループ図における S と O の意味を述べよ。

**問2** 次の連鎖が強化ループ(R)かバランスループ(B)か判定せよ:  
「ストレス↑ →(O) 睡眠↓ →(O) 疲労↑ →(S) ストレス↑」  
（リンク記号: S = 同方向変化、O = 逆方向変化）

**問3** システムダイナミクスにおけるストックとフローの違いを述べよ。

**問4** 「Fix That Fail（応急処置の失敗）」のシステム原型を、RループとBループの観点から簡潔に説明せよ。

### A-6 ファジィ・AHP

**問1** ファジィ理論と確率論の違いを、対象と意味の観点から述べよ。

**問2** AHPにおける一対比較と整合性指標 CR の役割を述べよ。

### A-7 使い分け（論述）

次の状況に適した手法（ISM / DEMATEL / Cognitive Map / AHP / SD）を選び理由を述べよ。

(a) SNSのフォロー関係に階層はないが、影響の強さを知りたい  
(b) 製品評価基準の下で3案の優先順位を決めたい  
(c) 要因Aが要因Bを「増やす／減らす」の符号が重要  
(d) 影響・依存関係が推移的で、上位・下位の階層が欲しい  
(e) 政策介入後10年間の人口変化を試算したい

## B. モデリング・シミュレーション（藤井担当）

### B-1 確率・マルコフ

**問1** 確率モデルと確率過程を定義せよ。

**問2** マルコフ性を条件付き確率の式で書き、言葉で説明せよ。

**問3** 推移行列

$$
P = \begin{pmatrix} 0.9 & 0.1 \\ 0.2 & 0.8 \end{pmatrix}
$$

の定常分布 $\boldsymbol{\pi}$ を求めよ（$\pi P = \pi$, $\sum \pi_i=1$）。

**問4** 分布関数 \(F(x)=P(X\leq x)\) の性質を3つ挙げよ。

### B-2 待ち行列

**問1** M/M/1モデルにおける利用率 \(\rho\) の定義と、\(\rho \geq 1\) のときの意味を述べよ。

**問2** リトルの法則 \(L=\lambda W\) の各記号の意味を述べよ。

### B-3 離散モデル・SOC

**問1** 有限オートマトンとセルオートマトンの違いを述べよ。

**問2** 2次元セルオートマトン（サンドパイル）の更新規則を3ステップで述べよ。

**問3** 自己組織化臨界（SOC）とは何か。サンドパイルとべき乗則の関係を1文で述べよ。

**問4** エージェントとマルチエージェントシステム（MAS）を定義し、創発の例を1つ挙げよ。

**問5** 連続体現象を離散モデル化する理由を2点述べよ。

### B-4 エージェント構造

**問1** 単純反射エージェントと効用ベースエージェントの違いを述べよ。
