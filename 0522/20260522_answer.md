## １．講義で理解できた内容、興味を持った内容

ファジィ理論は、確率が扱う事象が起きるかどうかの不確実性とは別に、言葉や概念のあいまいさ（fuzziness）を [0, 1] の所属度で表現する枠組みだと整理できた。社会科学では数値化しにくい情報が多く、主観を含む判断を定式化する必要性が説明され、第2回までの構造モデリング（ISM・DEMATEL）と並ぶあいまい問題への挑戦として位置づけられた点が印象的だった。

メンバーシップ関数について、クリスプ集合では境界が 0/1 だが、ファジィ集合では μ(x) が連続的に変化する、という対比が明確だった。暑い適温などの言語ラベルは、三角形や台形の標準形で近似できること、またグレードは割合ではなく主観に基づく順序情報だという説明は、後の演算の意味を理解するうえで重要だった。

ファジィ集合の演算は、論理演算に対応して和＝max、積＝min、補＝1−μ と覚えやすかった。講義の通勤時間の例では、短い OR 快適は max、短い AND 快適は min、短いが快適でないは A と B の補の min で計算でき、直感と式が一致した。

三角型ファジィ数 (a, b, c) では、b が最も確からしい値で、a・c が下限・上限になる。加減算はパラメータの足し引きでよく、α-カットで区間 [a+α(b−a), c−α(c−b)] に落とすと、不確実性の幅が α に応じて狭まる様子が視覚的に理解できた。組立時間と検査時間を足した総製造時間の例では、α=1.0 で一点 {55} 分に収束し、不確実性が消える極限の意味も掴めた。

ファジィ関係の max-min 合成 μ(x,z) = max_y min[μ_R1(x,y), μ_R2(y,z)] は、中間要素 y を介した間接的なつながりの強さを表すと説明され、第2回の可到達行列・グラフの考え方の延長として興味深かった。反射・対称・推移の違い、特にファジィでは推移律が単純でない点は、ISM の推移律前提との対比として記憶に残った。

講義後半のショケ積分は、加法モデルでは拾えないどちらか一方が優れていればよいという非加法的な評価基準を表現できる点に強い関心を持った。英会話・情報処理の例で、P君は加法平均 55 点・ショケ積分 83 点、Q君は両方 60 点となり、目的に応じて総合評価の定義を変える必要があることが具体的に分かった。

配布の `fuzzy_exercise_tool_3.html` と `fuzzy_ism_dematel_1.html` で手を動かし、式と数値結果を即座に照合できるのは復習に有効だと感じた。

## ２．理解できなかった内容、不明な点

ファジィ関係の推移律について、3要素だけで成立しても、別の経路を通ると成立しないという説明の厳密な判定手順がまだ曖昧である。講義では推移的閉包 R* を求める話もあったが、手計算の規模が大きい場合の実務的な打ち切り基準が分かっていない。

台形型ファジィ数の区間演算で、(P+Q)_α = P_α + Q_α が成り立つことは確認できたが、なぜ三角型のパラメータ直和と α-カット後の区間加算が一般に一致しないのか、理論的な条件（凸性・正規性など）まで理解できていない。

λ-ファジィ測度の λ の符号（加法的・相乗的・代替的）と、実データから λ を推定する手順は、スライド末尾で触れられたが、例題の c の求め方（高次方程式）まで追い切れなかった。

ファジィ ISM の α 閾値を変えると辺の本数が変わり階層が変わる、という挙動はツールで確認したが、どの α を採用するかの客観的な選び方（専家合意・感度分析の止め方）が不明である。

スライド Q1.4（ルート選択）では、区間の数値表示（35・40）と三角型ファジィ数の見え方が一枚絵で重なっており、各辺の (a,b,c) の読み取りと、ルート全体のファジィ和の対応関係を自力で再現するのに時間がかかった。講義中の板書または解答スライドの対応表があれば助かる。

## ３．とくに知りたいことや感想・要望など

第3回の因果ループ・システムダイナミクスで扱った曖昧な因果の強さを、FCM やファジィ DEMATEL とどう接続して一貫したモデルにするか、事例ベースでもう一段ほしい。特に、符号付きの CM と非負の DEMATEL のあいだをファジィ重みで埋める設計指針があると、期末までのレポート設計に活かせる。

ショケ積分と次回予定の AHP を組み合わせた評価フロー（基準の重みは AHP、基準間の相互作用はファジィ測度）の最小ワークフローを、1テーマ（例：製品選定）で通しで見たい。

演習ツールは非常に有用なので、試験範囲に含まれる計算（max-min 合成、α-カット、Choquet の 2 基準）について、紙で解く際の手順チェックリストがあると安心できる。

### 演習問題1 — Q3.1・Q3.2（p.93〜94）

男子 X={x₁,x₂}、女子 Y={y₁,y₂,y₃}。R＝英語の成績がほぼ同じ、S＝数学の成績がかなりよい。

R = [[0.3,0.6,0.1],[0.7,0.1,0.8]]、S = [[0.4,0.9,0.7],[0.5,0.6,0.2]]

**Q3.1** R∪S = [[0.4, **0.9**, 0.7],[0.7, 0.6, 0.8]]  
→ 最大は **x₁−y₂ で 0.9**（英語が同じ OR 数学がよい）。

**Q3.2** R∩S = [[0.3, **0.6**, 0.1],[0.5, 0.1, 0.2]]  
→ 最大は **x₁−y₂ で 0.6**（英語が同じ AND 数学がよい）。

### 演習問題2 — Q4.1・Q4.2（p.95〜96）

X={x₁,x₂,x₃}、Y={y₁,…,y₄}。R＝身長がほぼ同じ、S＝体重がほぼ同じ。

**Q4.1** R∪S = [[0.4,0.6,0.1,**0.9**],[0.1,0.4,0.7,0.6],[0.7,0.1,0.8,0.8]]
