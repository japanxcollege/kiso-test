## 必須公式

| 名称 | 式 |
|------|-----|
| 可到達（ISM） | \(M = A + I\), \(M_R \approx M^{n-1}\) で収束 |
| DEMATEL 正規化 | \(n_{ij} = x_{ij} / \max_k \sum_j x_{kj}\) |
| 総合影響 ⭐ | \(T = N(I-N)^{-1}\) |
| 影響度・関連度 ⭐ | \(D_i=\sum_j t_{ij}\), \(R_j=\sum_i t_{ij}\), 影響度=\(D-R\), 関連度=\(D+R\) |
| マルコフ性 ⭐ | \(P(X_{n+1}=j \mid X_n=i,\ldots)=P(X_{n+1}=j \mid X_n=i)\) |
| lステップ推移 | \(p_{ij}^{(l)} = (P^l)_{ij}\) |
| 定常分布 ⭐ | \(\boldsymbol{\pi} P = \boldsymbol{\pi}\), \(\sum \pi_i = 1\) |
| 分布関数 | \(F(x)=P(X\leq x)\) |
| 利用率（M/M/1） | \(\rho = \lambda/\mu\) |
| リトルの法則 | \(L = \lambda W\) |
| べき乗則 | \(N(x) = C x^{-\tau}\) |

## ISM 手順チェックリスト

1. 要素抽出  
2. 一対比較 → 構造行列（0/1）  
3. \(M=A+I\) のべき乗 → 可到達行列  
4. R, A 計算 → **R∩A=R** でレベル抽出  
5. 有向階層図・骨格行列  

---

## DEMATEL 手順チェックリスト

1. 直接影響評価（0〜4）  
2. 行列 X → 正規化 **N**  
3. **T = N(I−N)⁻¹**  
4. D=行和, R=列和  
5. **D−R** グラフ（原因系↑ 結果系↓）  

---

## マルコフ・待ち行列チェックリスト

1. 推移行列 P（各行の和=1）  
2. \(\pi P = \pi\) と \(\sum\pi_i=1\) を連立  
3. M/M/1: \(\rho=\lambda/\mu < 1\) で安定  

---

## サンドパイル（CA）チェックリスト

1. ランダムセル +1  
2. 値≥4 → −4、4近傍に+1（連鎖）  
3. 安定まで繰り返し

## 出てきそうなシーンと練習問題

上の各セクションに対応。詳解は [基本問題集.md](基本問題集.md) → [基本問題_回答集.md](基本問題_回答集.md)。

### 必須公式

| 対応 | 出てきそうなシーン | 練習問題 | 答え |
|------|-------------------|----------|------|
| 可到達 M=A+I | 隣接→階層化 | M²,M³ の意味と収束回数 | a_{ij}=1 は i→j の有向辺。**(A+I)^{n−1}** で可到達行列 |
| DEMATEL 正規化 | X→N | 行和最大値で除算する理由 | 級数 **N+N²+…** が収束するようスケール |
| 総合影響 T | **計算定番** | A-4 の X から N,T,D,R を求めよ | N=[[0,2/3,0],[0,0,1],[0,0,0]], T=[[0,2/3,2/3],[0,0,1],[0,0,0]] |
| D−R / D+R | 原因/結果 | A-4 の①②③を分類 | ①**原因**(+4/3), ③**結果**(−5/3) |
| マルコフ性 | 定義 | マルコフ性を式と言葉で | **P(X_{n+1}=j\|X_n=i,…)=P(X_{n+1}=j\|X_n=i)**。過去不要 |
| lステップ P^l | n ステップ後 | P=[[0.9,0.1],[0.2,0.8]] で 1→2（2step） | **0.17**（0.9×0.1+0.1×0.8） |
| 定常分布 | πP=π | 上の P の π | **π=(2/3, 1/3)** |
| 分布関数 F(x) | 公理 | F(x) の性質3つ | 単調非減少、右連続、F(−∞)=0,F(∞)=1 |
| 利用率 ρ | M/M/1 | ρ≥1 の意味 | **ρ=λ/μ**。ρ≥1 で待ち行列**発散・不安定** |
| リトルの法則 | 待ち行列 | L,λ,W の意味 | **L**=平均系内人数、**λ**=到着率、**W**=平均滞在時間 |
| べき乗則 | SOC | SOC とべき乗則 | サンドパイル崩壊規模が**べき乗分布**→SOCの指標 |

### 直前ミニセット（15分）

1. T=N(I−N)⁻¹ の N が [[0,0.5,0],[0,0,0.75],[0,0,0]] のとき T の (1,3) 成分だけ求めよ（A-4 類題）  
2. P=[[0.7,0.3],[0.4,0.6]] の定常分布  
3. ISM で R∩A=R の意味を2文  
4. S と O、R と B の判定ルールを1行ずつ  
5. M/M/1 で λ=9, μ=10 の ρ と安定性  

**答え**

1. **T(1,3)=0.375**（N から T=N(I−N)⁻¹ を計算）  
2. **π=(4/7, 3/7)**  
3. R=到達可能、A=先行。**R∩A=R** の要素を同レベルとして抽出  
4. **S**=同方向、**O**=逆方向。**R**=O偶数、**B**=O奇数  
5. **ρ=0.9<1** → **安定**
