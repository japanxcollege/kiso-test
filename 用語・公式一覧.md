## 1. 重要公式（暗記レベル）

### 構造化手法

| 名称 | 式 | 意味 |
|------|-----|------|
| 可到達行列 ⭐ | M = A + I、M^(n−1) で収束 | n ノードで n−1 回 |
| DEMATEL 正規化 | N = X / max(行和) | スペクトル半径 < 1 を保証 |
| 総合影響行列 ⭐ | T = N(I−N)⁻¹ = N + N² + N³ + … | N が冪零なら I + N + N² |
| 発信力 D | D_i = Σ_j t_ij（T の行和） | 他に与える総影響 |
| 被影響力 R | R_j = Σ_i t_ij（T の列和） | 他から受ける総影響 |
| 影響度 ⭐ | D − R | 正→原因系、負→結果系 |
| 関連度 ⭐ | D + R | システム内の中心性 |

### 因果ループ・SD

| 名称 | 式 / 規則 | 意味 |
|------|----------|------|

### 確率・マルコフ

| 名称 | 式 | 意味 |
|------|-----|------|
| 確率公理 | 0≤P≤1, P(Ω)=1, 可算加法性 | Kolmogorov |
| ベイズの定理 | P(A\|B) = P(B\|A)P(A)/P(B) | 条件付き確率 |
| マルコフ性 ⭐ | P(X_{n+1}=j \| X_n=i,…) = P(X_{n+1}=j \| X_n=i) | 過去履歴不要 |
| l ステップ推移 | p_ij^(l) = (P^l)_ij | 行列冪 |
| Chapman-Kolmogorov | P^(l+n) = P^l · P^n | 連結性 |
| 定常分布 ⭐⭐ | πP = π, Σπ_i = 1 | 平衡分布 |
| 分布関数 | F(x) = P(X ≤ x) | 単調非減少・右連続 |

### 待ち行列（M/M/1）

| 平均待ち行列長 ⭐ | L_q = ρ²/(1−ρ) | 窓口の人を除く |
| 平均滞在時間 | W = 1/(μ−λ) | |
| 平均待ち時間 | W_q = ρ/(μ−λ) | |
| リトルの法則 ⭐⭐ | L = λW | 分布によらない |

### 確率分布

| 分布 | 期待値 | 分散 |
|------|--------|------|
| ベルヌーイ(p) | p | p(1−p) |
| 二項 B(n, p) | np | np(1−p) |
| ポアソン(λ) | λ | λ |
| 幾何(p) | 1/p | (1−p)/p² |
| 指数(λ) ⭐ | 1/λ | 1/λ² |
| 正規 N(μ, σ²) | μ | σ² |

**無記憶性（指数分布）⭐**：P(X > s + t | X > s) = P(X > t)

### 複雑系・フラクタル

| 名称 | 式 | 意味 |

### スターリングの公式

**式**：n! ≈ √(2πn) (n/e)^n

## 3. ISM 手順チェックリスト

1. **要素抽出**（10 個程度）
2. **一対比較**で隣接行列 A（0/1）
3. **M = A + I の冪乗** で可到達行列を求める（n−1 回で収束）
4. **R(i), A(i) を計算**
5. **R ∩ A = R** の要素を最上位レベルとして抽出
6. **抽出した要素を除いて反復**
7. **骨格行列**で階層図を描く

---

## 4. DEMATEL 手順チェックリスト

1. **直接影響評価**（0〜4 の整数）→ X
2. **行和の最大値**で全体を割って **N**
3. **N², N³, … を計算**（N が冪零ならすぐ 0 になる）
4. **T = N + N² + N³ + … = N(I − N)⁻¹**
5. **D = T の行和**（発信力）
6. **R = T の列和**（被影響力）
7. **D − R グラフ**（原因系↑ 結果系↓）
8. **D + R**（関連度）でシステム中心性

---

## 5. マルコフ・待ち行列チェックリスト

1. **推移行列 P**（各行の和 = 1 を確認）
2. **πP = π** を成分ごとに書く
3. **Σπ_i = 1** を加える
4. **連立方程式を解く**
5. **検算**：πP を計算して π に戻るか確認

**M/M/1**：
1. **ρ = λ/μ**
2. **ρ < 1 で安定**
3. **L = ρ/(1−ρ)**（系内総数）
4. **L_q = ρ²/(1−ρ)**（待ち行列の長さ、窓口の人除く）

## 6. フラクタル次元チェックリスト

1. **問題図を見て、自己相似コピーの個数 N を数える**
2. **縮小率 r または拡大率 s = 1/r を読み取る**
3. **D = log N / log s** で計算
4. 「対数を小数に変換する必要はない」と書かれていれば **log N / log s** の形のままで OK

**典型図形の N と s**：
- シェルピンスキー：N=3, s=2
- カントール：N=2, s=3
- メンガー：N=20, s=3

## 7. オートマトン決定性化チェックリスト（部分集合構成法）

1. **初期状態 {q_0}** から始める
2. **各既存状態 S と各入力 a について**：
   - δ'(S, a) = ∪_{q ∈ S} δ(q, a) を計算
3. **新規状態なら追加**、繰り返し
4. **受理状態 F' = {S | S ∩ F ≠ ∅}**
5. **状態遷移表**または**状態遷移図**で答える

**Q と Q' の対応関係**も示す。

## 8. サンドパイル（CA）チェックリスト

2. **値 ≥ 4 のセル**から 4 減らし、4 近傍に +1
3. **連鎖崩壊**を 3 以下になるまで繰り返す

## 9. 因果ループ（CLD）

- **S（Same）**：同方向（A 増 → B 増）
- **O（Opposite）**：逆方向（A 増 → B 減）
- **R（強化ループ）**：ループ内の **O が偶数個**
- **B（バランスループ）**：ループ内の **O が奇数個**

**ループの積で判定**：S=+1, O=−1 として全体積が +1 → R、−1 → B
