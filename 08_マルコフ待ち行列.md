## 1. マルコフ連鎖

状態空間 \(S=\{s_1,\ldots,s_m\}\)。時刻 \(n\) の状態 \(X_n\)。

\[
P(X_{n+1}=j \mid X_n=i, X_{n-1},\ldots) = P(X_{n+1}=j \mid X_n=i) = p_{ij}
\]

**離散時間マルコフ連鎖** — 一般に「マルコフ連鎖」は一次を指す。

---

## 2. 推移確率行列 P ⭐

\[
P = (p_{ij}), \quad p_{ij} \geq 0, \quad \sum_{j=1}^{m} p_{ij} = 1 \text{（各行の和=1）}
\]

### l ステップ推移

\(p_{ij}^{(l)} = (P^l)_{ij}\) — **Chapman-Kolmogorov方程式**

\[
p_{ij}^{(l+n)} = \sum_k p_{ik}^{(l)} p_{kj}^{(n)}
\]

### 例: ギャンブラーの破産

所持金 \(i\) で +1（確率 \(p\)）/ −1（確率 \(q=1-p\)）。0 または N で終了 → 吸収状態あり。

## 3. 定常分布・極限分布 ⭐

### 定常分布 π

\[
\boldsymbol{\pi} P = \boldsymbol{\pi}, \quad \sum_i \pi_i = 1
\]

**平衡方程式** — 1ステップ後も分布が変わらない。

### 極限分布

\[
\boldsymbol{\pi} = \lim_{n\to\infty} \mathbf{p}^{(0)} P^n
\]

**既約かつ非周期的**なら極限分布＝定常分布（初期状態に依存しない）。

### 計算例（試験定番）

\[
P = \begin{pmatrix} 0.9 & 0.1 \\ 0.2 & 0.8 \end{pmatrix}
\Rightarrow \boldsymbol{\pi} = \left(\frac{2}{3}, \frac{1}{3}\right)
\]

手順: \(\pi_1+\pi_2=1\) と \(\pi P=\pi\) の連立。

## 4. 状態の分類（用語）

| 用語 | 意味 |
|------|------|
| **既約** | 全状態が相互到達可能 |
| **再帰的** | 有限時間で戻ってこれる |
| **非周期的** | 周期が1 |
| **吸収状態** | \(p_{ii}=1\)（そこから出られない） |
| **強マルコフ性** | 任意の再開時刻以降、過去と独立 |

## 5. 待ち行列理論

### 基本モデル M/M/1

- 到着: ポアソン（率 \(\lambda\)）
- サービス: 指数（率 \(\mu\)）
- 1サーバ、無限待ち行列

### 利用率（閉塞率）⭐

\[
\rho = \frac{\lambda}{\mu} = \frac{T_s}{T_a}
\]

- \(\rho < 1\) で安定
- \(\rho \to 1\) で待ち時間が急増

### リトルの法則

\[
L = \lambda W
\]

系内の平均顧客数 \(L\) = 到着率 × 平均滞在時間 \(W\)

### 拡張（名前だけ）

- **M/M/1/N**: 系の容量上限 N
- 直列・並列の M/M/1 系
