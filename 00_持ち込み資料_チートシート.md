## ⚡ 公式・手順の超圧縮版

### DEMATEL（最頻出計算）

📚 **原本**：[第02回スライド](講義スライド一覧/第02回_スライド.pdf) ・ [構造化レポート](第04回_構造化レポート演習/構造化レポート.pdf) ・ [Excel ツール](第04回_構造化レポート演習/ISM-DEMATEL計算ツール_A.xlsm)

```
1. 直接影響行列 X が与えられる（0〜4 で評価）
2. 各行の行和 → 最大行和を求める → max
3. N = X / max（正規化）
4. (I−N)⁻¹ = I + N + N² + N³ + ...
   → N が冪零（対角 0 の上三角等）なら I + N + N² で OK
5. T = N(I−N)⁻¹ = N + N² + N³ + ...
6. D = T の各行の和（発信力）
7. R = T の各列の和（被影響力）
8. 影響度 D−R：正なら原因系、負なら結果系
9. 関連度 D+R：システム内の中心性
```

**詳細・計算例** → [02_計算問題テンプレ Section 1](02_持ち込み資料_計算問題テンプレ.md)

### マルコフ定常分布

📚 **原本**：[第08回 SE Basics 5](講義スライド一覧/第08回_マルコフ待ち行列_System_Engineering_Basics_5.pdf) ・ [第06回 SE Basics 3](講義スライド一覧/第06回_確率モデル_System_Engineering_Basics_3.pdf)

```
1. 推移行列 P が与えられる（各行の和 = 1 を確認）
2. πP = π を成分ごとに書く
3. Σπ_i = 1 を加える
4. 連立方程式を解く

例：P = [[0.9, 0.1], [0.2, 0.8]]
  0.9π₁ + 0.2π₂ = π₁ → 0.1π₁ = 0.2π₂ → π₁ = 2π₂
  π₁ + π₂ = 1 → 3π₂ = 1 → π₂ = 1/3, π₁ = 2/3
  ∴ π = (2/3, 1/3)
```

### M/M/1

📚 **原本**：[第08回 SE Basics 5](講義スライド一覧/第08回_マルコフ待ち行列_System_Engineering_Basics_5.pdf)

```
利用率 ρ = λ/μ
ρ < 1 で安定、ρ ≥ 1 で発散
リトルの法則 L = λW（系内平均 = 到着率 × 平均滞在時間)
平均系内人数 L = ρ/(1−ρ)
平均滞在時間 W = 1/(μ−λ)
```

### S/O と R/B 判定

📚 **原本**：[第03回 因果SD](第03回_因果SD_原本/第03回_システム工学基礎.pdf)

```
S = +1（同方向変化）
O = −1（逆方向変化）
ループ内の積：
  +1 → R（強化ループ、O 偶数個）
  −1 → B（バランスループ、O 奇数個）
```

### ISM 階層抽出

📚 **原本**：[第02回スライド](講義スライド一覧/第02回_スライド.pdf) ・ [構造化レポート](第04回_構造化レポート演習/構造化レポート.pdf)

```
R(i) = 到達可能集合（i から到達できる要素）
A(i) = 先行集合（i に到達できる要素）
R(i) ∩ A(i) = R(i) なら i は最上位レベル
→ 抽出して除外、残りで反復
```

### 可到達行列

📚 **原本**：[第02回スライド](講義スライド一覧/第02回_スライド.pdf) ・ [構造化レポート](第04回_構造化レポート演習/構造化レポート.pdf)

```
隣接行列 A → M = A + I → M^(n−1) で可到達行列
n 個のノードでは n−1 回で収束
M^k = M^(k+1) になった時点で完了
```

## ⚡ 誤りを指摘 — 典型2パターン

### パターン1：「DEMATEL は因果の正負を + と − で表す」

> 誤り。DEMATEL は**非負の影響の強さ（0〜4）**を扱う。正負の符号を扱うのは **Cognitive Map（CM）** や **FCM**。DEMATEL の出力である D−R は正負を取るが、これは「原因系か結果系か」の判別であり、要素間の因果の符号ではない。

### パターン2：「マルコフ連鎖では過去の全履歴が次の状態を決める」

> 誤り。マルコフ性により、**現在の状態のみ**が次の状態を決定する。過去の履歴は不要。式で書けば：
> 
> P(X_{n+1}=j | X_n=i, X_{n-1}, X_{n-2}, ...) = P(X_{n+1}=j | X_n=i)
> 
> これにより推移行列 P だけで未来が予測でき、定常分布 πP=π が解析可能になる。

### その他想定される誤り

- 「ISM は影響の強さを扱う」→ 誤り。ISM は**0/1関係のみ**。強さは DEMATEL
- 「ρ ≥ 1 でも M/M/1 は安定」→ 誤り。**発散**する
- 「ファジィは確率と同じ」→ 誤り。確率は**発生不確実**、ファジィは**概念の曖昧さ**
- 「定常分布と極限分布は常に同じ」→ 注意。**既約 + 非周期的**で一致

## ⚡ 対比表（よく出る5セット）

### ISM vs DEMATEL

| | ISM | DEMATEL |
|---|---|---|
| 関係の表現 | 0/1（あり/なし） | 0〜4（強さ） |
| 推移律 | 仮定する | 仮定しない |
| 主目的 | 階層化 | 影響度・原因/結果の定量 |
| 出力 | 階層図 | D−R / D+R グラフ |

### 確率 vs ファジィ

| | 確率 | ファジィ |
|---|---|---|
| 対象 | 事象の**発生**の不確実性 | 概念の**曖昧さ**・程度 |
| 例 | 雨が降る確率 60% | 「高齢」の度合い 0.7 |
| 数学的基盤 | Kolmogorov 公理 | メンバーシップ関数 [0,1] |

### ストック vs フロー

| | ストック | フロー |
|---|---|---|
| 意味 | 時点で測れる蓄積量 | 単位時間あたりの増減 |
| 例 | 人口、貯金、CO₂濃度 | 出生率、入金、排出率 |
| 性質 | 過去の蓄積を持つ、慣性を生む | ストックを変化させる |

### R ループ vs B ループ

| | R（強化） | B（バランス） |
|---|---|---|
| 挙動 | 自己強化・発散 | 均衡・収束 |
| O 個数 | 偶数 | 奇数 |
| 例 | 貯金 → 利息 → 貯金 | 体温 → 発汗 → 体温 |

### 構造モデリング SM vs 動的モデリング DM

| | SM | DM |
|---|---|---|
| 焦点 | 関係・階層（静的） | 時間変化・フィードバック（動的） |
| 手法 | ISM, DEMATEL, CM, AHP | SD（システムダイナミクス） |
| 問い | 何が何に影響するか | 時間とともにどう変わるか |

## ⚡ 手法の使い分け（5択問題対策）

| 状況 | 答え | 理由 |
|------|------|------|
| 階層が知りたい、関係は推移的 | **ISM** | 推移律と階層抽出 |
| 影響の強さ、原因/結果を知りたい | **DEMATEL** | 0〜4 評価、D−R |
| 正負の因果が本質 | **Cognitive Map** | 符号で表現 |
| 代替案からの選択 | **AHP** | 一対比較で重み |
| 時間変化を試算したい | **SD** | ストック/フロー |
| 状態遷移の不確実性 | **マルコフ** | 確率的状態遷移 |
| 待ち時間・サービス | **待ち行列（M/M/1）** | ρ < 1 で安定 |
| 局所ルールから大域へ | **CA / エージェント** | 創発 |
| 連続現象の数値解 | **PDE → 離散化** | FDM/FEM |

## ⚡ 試験当日の動き方

1. **試験開始 → 全問題を 3 分眺める**：配点と問題タイプを把握
2. **計算問題から着手**（DEMATEL、マルコフ）：これで 18 点前後を確保
3. **このチートシートの該当箇所を参照しながら計算**：暗記不要
4. **論述は用語説明テンプレ（4段構成）を見ながら書く**
