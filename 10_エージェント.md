## 1. エージェントの定義

**環境を知覚し、行動する自律的実体。**

- センサ（S）で環境を知覚
- アクチュエータ（A）で環境に作用
- 環境の**一部**しか知覚・制御できない

### 例

| 種類 | 例 |
|------|-----|
| 物理 | 自律ロボット |
| ソフトウェア | 検索エージェント、市場参加プログラム |

---

## 2. 環境の分類（試験用語）

| 軸 | 区分 |
|----|------|
| 観測可能性 | 完全可観測 / 部分可観測 |
| 決定性 | 確定的 / 確率的 / 戦略的 |
| エピソード | エピソード的（各行動が独立）/ 連続的 |
| 時間 | 静的 / 半動的 / 動的 |
| 離散性 | 離散的 / 連続的 |
| エージェント数 | シングル / **マルチ** |

### マルチエージェント（MAS）

- **競争的**: 自分最大化・相手最小化
- **協力的**: 全体の性能指標最大化
- 例: 市場参加エージェント → **市場全体のダイナミクス**（創発）

---

## 3. エージェントの種類

| 種類 | 特徴 |
|------|------|
| ソフトウェアエージェント | プログラムとして自律動作 |
| 知的エージェント | 推論・学習能力 |
| インタフェースエージェント | UI支援 |
| モバイルエージェント | ネットワーク上を移動（可動性） |

---

## 4. エージェントの構造（能力の段階）⭐

| レベル | 内容 |
|--------|------|
| **単純反射** | 現在の知覚 → 行動（条件付き反射） |
| **モデルベース反射** | 内部状態で部分可観測に対応 |
| **目的ベース** | ゴールに向けた行動選択 |
| **効用ベース** | 効用関数の期待値最大化（合理的エージェント） |

---

## 5. 複雑系・複雑適応系との接続

### 複雑系

局所相互作用 → 大域的振る舞いが個々からは予測困難。

代表モデル: **セルオートマトン**、**マルチエージェントシステム**

### 複雑適応系（CAS）

構成要素が状況に応じて振る舞いを変化。

**ミクロ・マクロ・ループ**:
- ミクロ（局所）→ マクロ（全体）の創発
- マクロ → ミクロへのフィードバック

例: 遺伝子ネットワーク、脳、**マーケット**
