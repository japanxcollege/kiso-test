# システム工学基礎 — 試験まで12時間の勉強計画

試験まで残り 12 時間と仮定したときの**やる順番と所要時間**。

## 戦略の前提

この科目の試験は：

- 出席・レポート **60点** + 期末試験 **40点**
- 持ち込み**なし**、講義全体から出題
- 計算問題（DEMATEL、定常分布）と論述（用語説明、対比、誤り指摘）が中心

**12時間の使い方の原則**：

1. **計算問題を完璧に**（DEMATEL と マルコフ定常分布で約 10〜15 点）
2. **論述の頻出パターン**を抑える（用語説明、ISM vs DEMATEL、S/O判定、SOC）
3. **誤りを指摘型**の準備（DEMATEL の符号、マルコフ性の意味など）
4. **睡眠を削らない**：12 時間のうち最低 6 時間は睡眠と休憩

→ 実質**勉強時間は 6〜8 時間**として計画する。

#### ② DEMATEL の全計算を手で1回やる（45分）

ここが本科目で最も得点しやすい計算問題。基本問題 A-4 を**何も見ずに**解けるまで反復する。

手順を覚える：

1. **直接影響行列 X**（0〜4 で評価された生データ）
2. **正規化**：各行の行和を計算 → 最大行和で全体を割る → **N**
3. **(I−N)⁻¹ を計算**：N が冪零なら **I + N + N²** で OK
4. **T = N(I−N)⁻¹** を計算
5. **D = 各行の和**、**R = 各列の和**
6. **D+R（関連度）**、**D−R（因果性）** を計算
7. **D−R > 0 → 原因系**、**D−R < 0 → 結果系**

具体例（基本問題 A-4）：

X = [[0,2,0],[0,0,3],[0,0,0]]

- 行和は (2, 3, 0)、最大は 3
- N = [[0, 2/3, 0], [0, 0, 1], [0, 0, 0]]
- N² = [[0, 0, 2/3], [0, 0, 0], [0, 0, 0]]
- N³ = 0
- (I−N)⁻¹ = I + N + N² = [[1, 2/3, 2/3], [0, 1, 1], [0, 0, 1]]
- T = N + N² = [[0, 2/3, 2/3], [0, 0, 1], [0, 0, 0]]
- D = (4/3, 1, 0)、R = (0, 2/3, 5/3)
- ① D−R = +4/3（最も原因系）
- ③ D−R = −5/3（最も結果系）

**ポイント**：N が冪零（上三角でかつ対角が0）のとき、(I−N)⁻¹ = I + N + N² で済むことを覚える。手計算でつまずきがちな逆行列計算が一気に楽になる。

#### ③ マルコフ定常分布の計算を1回やる（30分）

これも頻出。

手順：

1. 推移行列 **P が与えられる**（各行の和 = 1 を確認）
2. **πP = π** を成分ごとに書き下す
3. **Σπ_i = 1** を加える
4. 連立方程式を解く

具体例：P = [[0.9, 0.1], [0.2, 0.8]]

- 0.9π₁ + 0.2π₂ = π₁ → 0.1π₁ = 0.2π₂ → π₁ = 2π₂
- π₁ + π₂ = 1 → 2π₂ + π₂ = 1 → π₂ = 1/3
- よって **π = (2/3, 1/3)**

別の例：P = [[0.7, 0.3], [0.4, 0.6]] → **π = (4/7, 3/7)**

#### ④ M/M/1 の利用率 ρ を整理（15分）

- **ρ = λ/μ**（到着率÷サービス率）
- ρ < 1 で**安定**、ρ ≥ 1 で**発散**
- **リトルの法則 L = λW**（系内平均人数 = 到着率 × 平均滞在時間）

λ=8, μ=10 → ρ=0.8 < 1（安定）
λ=9, μ=10 → ρ=0.9（安定だが待ち長い）

#### ⑦ 第2回 グラフ・ISM・DEMATEL（60分）

ここは試験の中心。試験総合対策.md の第2回パートを精読。

**必ず言えるべきこと**：

- **隣接行列 a_{ij}** の意味：i → j に有向エッジがある
- **M = A + I の冪乗で可到達行列**：n ノードでは n−1 回で収束
- **推移律**：iRj かつ jRk なら iRk
- 推移律が成り立たない例：友人関係、じゃんけん
- **R(i)：到達可能集合**、**A(i)：先行集合**
- **R∩A=R の意味**：i から下流に出られない＝最上位レベル
- **階層抽出アルゴリズム**：最上位を抽出→除外して反復
- **ISM vs DEMATEL の4点比較**：関係（0/1 vs 0〜4）、推移律（あり vs なし）、入力（一対比較 vs アンケート）、目的（階層化 vs 影響度）
- **D = 行和（発信力）、R = 列和（被影響力）**
- **D+R = 関連度（中心性）、D−R = 因果性（原因/結果系）**

ここで、もう一度 DEMATEL の計算を**白紙から**やってみる（時間制限 10 分）。

#### ⑧ 第3回 因果ループ・SD（45分）

**必ず言えるべきこと**：

- **S（同方向）、O（逆方向）**
- **R（強化）= O 偶数個、B（バランス）= O 奇数個**
- **判定の数学的根拠**：S=+1, O=−1 として全体積を取る
- **ストック vs フロー**：蓄積量 vs 単位時間変化
- ストックは過去の蓄積を持つ → **慣性・遅延**を生む
- **Fix That Fails**：B（即効）+ R（副作用）の典型システム原型

「ストレス↑ →(O) 睡眠↓ →(O) 疲労↑ →(S) ストレス↑」のような例題を3つ自分で作り、R/B を判定する練習。

#### ⑨ 第4-5回 ファジィ・AHP（25分）

- **AHP は代替案選択用**：階層構造 + 一対比較 + CR<0.1

### T−7h 〜 T−4h：確率・離散の論述系（3時間）

**目的**：第6〜12回（藤井担当）の論述問題に書ける状態にする。

#### ⑪ 第6-7回 確率の基礎と分布（30分）

- **確率公理 3つ**：0≤P≤1、P(Ω)=1、可算加法性
- **確率モデル vs 確率過程**：一般 vs 時間とともに変化
- **マルコフ性の式**：

  $$P(X_{n+1}=j | X_n=i, X_{n-1}, \ldots) = P(X_{n+1}=j | X_n=i)$$

- 言葉で：「**次の状態は現在の状態だけで決まり、過去の履歴は不要**」
- **分布関数 F(x) = P(X≤x)** の3性質：単調非減少・右連続・F(±∞)=0,1
- **代表分布6種**（ベルヌーイ、二項、ポアソン、幾何、指数、正規）の用途
- **指数分布の無記憶性**：P(X>s+t | X>s) = P(X>t) → マルコフ性と同型

#### ⑫ 第8回 マルコフ連鎖と待ち行列（45分）

- **推移確率行列 P**：各行の和 = 1
- **Chapman-Kolmogorov 方程式**：P^(l+n) = P^l · P^n
- **定常分布 π**：πP = π かつ Σπ_i = 1
- **既約 + 非周期的**なら**極限分布 = 定常分布**
- **M/M/1 の主要公式**：L = ρ/(1−ρ)、W = 1/(μ−λ)
- **リトルの法則 L = λW**：分布によらず常に成立

定常分布計算をもう一度やる（時間制限 10 分）。

#### ⑬ 第9-10回 オートマトン・CA・エージェント（45分）

- **有限オートマトン vs セルオートマトン**：1 体の決定的系 vs 多体の局所ルール
- **CA の更新**：局所ルールで**同時更新**
- **Game of Life の規則**：生存（隣 2 or 3）、誕生（隣ちょうど 3）
- **エージェントの5タイプ**：単純反射→モデルベース→目標→効用→学習
- **MAS（マルチエージェントシステム）**：集中制御なし、相互作用
- **創発**：局所ルールから大域の予測困難な性質
- **創発の例**：市場価格、boids、Schelling の分居モデル、アリのフェロモン

#### ⑭ 第11回 SOC とべき乗則（30分）

ここは試験で書けると得点が伸びる。

- **複雑系の特徴**：べき乗則・フラクタル・1/f ゆらぎ
- **自己組織化臨界（SOC）**：外部調整なしに臨界状態を保つ。Bak 1987
- **サンドパイル・モデル**の3ステップ更新：①ランダム +1 ②値≥4 で4近傍に拡散 ③連鎖崩壊
- **べき乗則**：N(x) ∝ x^(−τ)、両対数で直線
- **べき乗の現実例**：地震（Gutenberg-Richter）、所得（パレート）、都市規模（Zipf）
- **正規分布 vs べき乗分布**：細い裾 vs 太い裾（ブラック・スワン）

#### ⑮ 第12回 連続体の離散化（20分）

- 離散化の理由：①コンピュータは離散計算のみ ②PDE を差分方程式に置き換え
- **Navier-Stokes 方程式**：流体の連続体方程式
- **離散化手法4種**：FDM（差分）、FEM（基底関数）、FVM（体積保存）、粒子法

| フェーズ | 時間 | 内容 |
|---------|------|------|
| **T−12h 〜 T−10h** | 2h | 俯瞰 + DEMATEL/マルコフの計算徹底 |

### 重要原則

1. **計算問題は完璧にする**：DEMATEL と定常分布で**確実に**点を取る
2. **論述は「定義 → なぜ重要 → 具体例 → 関連概念」の4段**で書く
3. **対比とよく混同する語**を一通り頭に入れる
4. **休憩を取る**：2 時間に 1 回は 10〜15 分の休憩
5. **直前は新しいことを覚えない**：知っていることを確認するだけ

### 残り時間が少ないときの優先順位

もし 12 時間ではなく 6 時間しかなければ、次の順に縮める：

1. DEMATEL 計算（必ず）
2. マルコフ定常分布（必ず）
3. ISM の R∩A=R 階層抽出（必ず）
4. R/B 判定、システム3要素、SM/DM の違い（論述頻出）
5. 「よく混同する語」10 個程度（誤り指摘対策）

3 時間しかなければ、計算 2 つ（DEMATEL + 定常分布）と、用語・公式一覧.md の手順チェックリストだけに集中する。
