## 2. 確率モデル・確率過程

### 確率モデル

**確率的なルール**で事象をモデル化したもの。  
システム（または構成要素）の**状態**が確率変数となり、確率分布に従って変化する。

### 確率過程

状態が確率変数として**時間とともに変化**する過程。  
（講義では単一変量を中心に説明）

## 3. 確率の基礎（用語）

| 用語 | 意味 |
|------|------|
| 試行 trial | 結果が偶然で予測不能な実験・観察 |
| 標本空間 Ω | すべての結果の集合 |
| 事象 event | Ω の部分集合 |
| 事象の族 ℱ | 和・積・補に閉じた部分集合族 |
| 確率 P(A) | 0≤P(A)≤1, P(Ω)=1, 可算加法性 |

### 重要な性質

- 余事象: \(P(A^c) = 1 - P(A)\)
- 和の法則: \(P(A \cup B) = P(A) + P(B) - P(A \cap B)\)
- 単調性: \(A \subset B \Rightarrow P(A) \leq P(B)\)

---

## 4. マルコフ性 ⭐

\[
P(X_{n+1}=j \mid X_n=i, X_{n-1},\ldots,X_0) = P(X_{n+1}=j \mid X_n=i)
\]

**未来は現在の状態だけで決まり、過去の履歴は不要。**

### 直観例

- ランダムウォーク: \(S_{n+1}\) の予測に \(S_n\) のみ必要
- ギャンブラーの破産問題: 現在の所持金だけで次の勝敗が決まる

一次マルコフ連鎖が標準。二次マルコフ連鎖は \(X_n, X_{n-1}\) に依存。

---

## 5. 条件付き確率・ベイズ

\[
P(A \mid B) = \frac{P(A \cap B)}{P(B)}, \quad P(B)>0
\]

ベイズの定理: \(P(A|B) P(B) = P(B|A) P(A)\)
