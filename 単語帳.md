# システム工学基礎 — 単語帳

試験の「用語説明」「対比」「使い分け」に対応した語彙集。公式・計算手順は [用語・公式一覧.md](用語・公式一覧.md)、概念の流れは [試験総合対策.md](試験総合対策.md) を参照。

## 使い方

1. **回ごと**に見出しを追う（講義順）
2. 各語の「**一言意味**」を口に出してから「**試験メモ**」を確認
3. 「**対比**」はセットで覚える（例：ISM vs DEMATEL）
4. 直前は「よく混同する語」セクションで弱点チェック

記号：⭐＝高頻度・必須、🔗＝他用語とセットで覚える

## 第1回　ガイダンス — システムの定義と問題の分類

| 用語 | 意味 | 試験メモ・対比 |
|------|------|----------------|
| **システム** ⭐ | **要素・相互作用・目的性**を持つ全体 | 3要素セットで言える。「要素だけ」「相互作用だけ」ではシステムではない |
| 要素 | システムを構成する個々の実体 | 人・物・情報・組織単位など |
| 相互作用 | 要素間の関係・因果・物質や情報の流れ | これがあって初めてシステム |
| 目的性 | システム全体が達成する機能・目標 | 「何のためか」 |
| **境界** ⭐ | システム内外の区切り | 境界の引き方が分析の出発点 |
| 存在論的境界 | **客観世界に実在する**区切り | 細胞膜、国境、法人格 |
| 認識論的境界 | **分析者がモデル化のために設定**する区切り | 同じ対象でも観点で変わる |
| **良定義問題** | 目標・手段・評価基準が明確 | 最適化・シミュレーションで解ける |
| **悪定義問題（wicked problem）** ⭐ | 目的・関係者・解が曖昧 | 構造化手法（ISM等）が有効。Rittel & Webber 1973 |
| ソフトシステムズ方法論 | Checkland。世界観（Weltanschauung）を引き出しながら学習 | 悪定義問題への代表的アプローチ |
| CATWOE | Customer, Actor, Transformation, Worldview, Owner, Environmental constraints | ソフトシステムズの問題定義 |
| **SM（構造モデリング）** ⭐ | 関係・階層を**静的**に整理 | ISM, DEMATEL, CM, AHP |
| **DM（動的モデリング）** ⭐ | 時間変化・フィードバックを扱う | SD（システムダイナミクス） |
| SoS（System of Systems） | 独立したシステムが連携した上位の系 | 運用独立・管理独立・地理分散・創発・進化の5特性 |
| V-Model | 要求→設計→実装 ↔ 検証・評価 が対応する設計プロセス | 設計学基礎との対比 |
| 6つの認識側面 | 技術・人・組織・経済・政治・倫理 | システムを多面的に捉える |
| システムズアプローチ | 問題把握→構想→試行→改善の反復 | プロセス全体の名前 |

## 第2回　グラフ・ISM・DEMATEL — 構造を行列で扱う

### 行列とグラフの基礎

| 用語 | 意味 | 試験メモ |
|------|------|----------|
| **隣接行列 A** ⭐ | a_{ij} = 1 で i→j のエッジ | 有向は非対称 |
| 行列の冪 A^k | (i,j) 成分 = i から j への**長さ k の経路の本数** | k 乗で何が分かるか |
| **M = A + I** ⭐ | 自己ループを加えた隣接行列 | 「長さ 0〜k の経路」を扱う |
| **可到達行列** ⭐ | M^(n−1) で得られる「到達可能か」の表 | n−1 回で収束（推移律下） |
| ブロック三角化 | 強連結成分をまとめ階層順に並べ替え | ISM の前処理 |
| 骨格行列 | A→B→C のとき派生する A→C を除いた**最小構造** | 描画の可読性向上 |

### ISM 関連

| 用語 | 意味 | 試験メモ |
|------|------|----------|
| **ISM**（Interpretive Structural Modeling）⭐ | 推移的依存関係から階層構造を抽出 | Warfield 1973 |
| **推移律** ⭐ | iRj かつ jRk ならば iRk | ISM の前提 |
| 推移律の反例 | 友人関係、じゃんけん、相性 | 全関係が推移的とは限らない |
| **R(i)：到達可能集合** ⭐ | i から到達できる要素の集合 | 「i の下流」 |
| **A(i)：先行集合** ⭐ | i に到達できる要素の集合 | 「i の上流」 |
| **R(i) ∩ A(i) = R(i)** ⭐ | i が**最上位レベル**にある条件 | i から下流に出られない＝最上位 |
| 階層抽出アルゴリズム | 最上位を抽出→除外して反復 | 順次レベルを決定 |

### DEMATEL 関連

| 用語 | 意味 | 試験メモ |
|------|------|----------|
| **DEMATEL** ⭐ | 影響の強さを 0〜4 で評価し、間接影響も含めて集計 | Battelle Geneva 1972 |
| **直接影響行列 X** | 専門家が 0〜4 で評価した生データ | 0=なし、4=非常に強い |
| **正規化行列 N** ⭐ | X を最大行和で割る | 級数収束のため |
| **総合影響行列 T = N(I−N)⁻¹** ⭐ | N + N² + N³ + … の総和 | 直接 + 間接 |
| **D（行和）** ⭐ | 要素 i が他に与える総影響量 | 発信力 |
| **R（列和）** ⭐ | 要素 i が他から受ける総影響量 | 被影響力 |
| **D + R（関連度）** ⭐ | システム内での重要性 | 中心性 |
| **D − R（因果性）** ⭐ | 正なら原因系、負なら結果系 | 原因系に介入 |
| 原因系 | D−R > 0 | 介入すべき根本要素 |
| 結果系 | D−R < 0 | 対症療法の対象 |

## 第3回　因果ループ・SD — 時間とフィードバック

### 因果ループ図

| 用語 | 意味 | 試験メモ |
|------|------|----------|
| **Cognitive Map** ⭐ | 符号付き因果グラフ | Axelrod 1976 |
| 符号 +/− | 因果の方向（正/負） | DEMATEL と異なり**強さは扱わない** |
| 不確定 | 複数パスで符号が矛盾 | CM の限界 |
| **因果ループ図（CLD）** ⭐ | S/O 矢印で因果と符号を可視化 | SD の質的版 |
| **S（Same）** ⭐ | 同方向変化（A 増 → B 増） | +1 として扱う |
| **O（Opposite）** ⭐ | 逆方向変化（A 増 → B 減） | −1 として扱う |
| **強化ループ R** ⭐ | 自己強化・発散 | O が偶数個 |
| **バランスループ B** ⭐ | 均衡・収束 | O が奇数個 |
| ループ判定 | S=+1, O=−1 とした全体積で判定 | 積 +1 → R、−1 → B |

### システム原型（Senge）

| 用語 | 意味 |
|------|------|
| システム原型 | SD でよく出る典型パターン |
| **Fix That Fails** ⭐ | 即効の対症療法（B）+ 副作用の悪化（R） |
| Shifting the Burden | 症状対処に頼って根本悪化 |
| Limits to Growth | 成長ループに必ず制約ループが出る |
| Tragedy of the Commons | 個人合理性が共有資源を破壊 |
| Eroding Goals | 目標が現実に合わせて低下 |
| Success to the Successful | 強者がさらに強くなる |
| Escalation | 報復の連鎖でエスカレート |
| Growth and Underinvestment | 投資不足で成長が止まる |

|------|------|----------|
| **SD** ⭐ | フィードバックループとストック/フローで時間発展をモデル化 | Forrester 1961 |
| **ストック** ⭐ | 時点で測れる蓄積量（状態変数） | 人口、貯金、CO₂濃度 |

## 第5回　AHP — 多基準意思決定

| 用語 | 意味 | 試験メモ |
|------|------|----------|
| **AHP** ⭐ | 階層構造 + 一対比較で代替案を選ぶ | Saaty 1980 |

| 用語 | 意味 | 試験メモ |
|------|------|----------|
| 試行 | 結果が偶然に決まる実験 | コイン、サイコロ |

## 第7回　確率分布

| 用語 | 意味 | 試験メモ |
|------|------|----------|
| **分布関数 F(x) = P(X ≤ x)** ⭐ | 累積分布関数（CDF） | 単調非減少・右連続・F(±∞)=0,1 |
| 確率質量関数 p(x) | 離散型の確率 | P(X=x) |
| 確率密度関数 f(x) | 連続型の密度 | ∫f=1、P(X=x)=0 |
| **期待値 E[X]** | 平均値 | 線形性 E[aX+b]=aE[X]+b |
| 分散 V[X] | (X − E[X])² の期待値 | バラつき |
| ベルヌーイ分布 | 1 回の成功/失敗 | パラメータ p |
| **二項分布 B(n,p)** | n 回独立試行の成功数 | E=np, V=np(1−p) |
| ポアソン分布 | 単位時間の稀な事象数 | E=λ, V=λ |
| 幾何分布 | 初成功までの試行数 | E=1/p |
| **指数分布** ⭐ | 待ち時間（無記憶性） | E=1/λ |
| **無記憶性** ⭐ | P(X>s+t \| X>s) = P(X>t) | マルコフ性と同型 |
| **正規分布 N(μ,σ²)** | 測定誤差・中心極限定理 | 自然現象 |
| 中心極限定理 | 独立同分布の和は正規に収束 | 統計の基礎 |
| **モンテカルロ法** ⭐ | 乱数サンプルで期待値近似 | 誤差 ∝ 1/√N |

## 第8回　マルコフ連鎖と待ち行列

### マルコフ連鎖

| 用語 | 意味 | 試験メモ |
|------|------|----------|
| マルコフ連鎖 | マルコフ性を持つ離散時間確率過程 | 状態空間 S |
| **推移確率 p_{ij}** ⭐ | 状態 i から j への 1 ステップ遷移確率 | |
| **推移確率行列 P** ⭐ | (p_{ij}) を並べた行列 | 各行の和 = 1 |
| l ステップ推移 | p_{ij}^{(l)} = (P^l)_{ij} | 行列冪 |
| **Chapman-Kolmogorov 方程式** | P^(l+n) = P^l · P^n | 連結性 |
| **定常分布 π** ⭐ | πP = π、Σπ_i = 1 | 平衡分布 |
| 極限分布 | lim_{n→∞} p^(0) P^n | 既約+非周期で π と一致 |
| **既約**（irreducible） | 全状態が相互到達可能 | 連結性 |
| **再帰的**（recurrent） | 有限時間で戻る | |
| 過渡的（transient） | 戻らない確率が正 | |
| **非周期的**（aperiodic） | 周期が 1 | 戻り間隔がバラバラ |
| 吸収状態 | p_ii = 1 | 破産問題 |
| 強マルコフ性 | 停止時刻以降、過去と独立 | |

### 待ち行列

| 用語 | 意味 | 試験メモ |
|------|------|----------|
| 待ち行列理論 | 到着・サービス過程のモデル | Erlang 1909 |
| ケンドール記法 A/B/c | 到着/サービス/サーバ数 | M/M/1 |
| **M/M/1** ⭐ | ポアソン到着・指数サービス・1サーバ | 基本モデル |
| **利用率 ρ = λ/μ** ⭐ | サーバが忙しい時間の割合 | ρ<1 で安定 |
| 安定条件 | ρ < 1 | ρ≥1 で発散 |
| **リトルの法則 L = λW** ⭐ | 系内平均人数 = 到着率 × 平均滞在時間 | 分布によらず成立 |
| L_q（待ち行列長） | 系内ではなく待ち行列の長さ | ρ²/(1−ρ) |
| W_q（待ち時間） | サービス開始までの時間 | ρ/(μ−λ) |
| M/M/c | サーバ c 台 | Erlang C 公式 |
| M/G/1 | サービスが一般分布 | Pollaczek-Khinchine |
