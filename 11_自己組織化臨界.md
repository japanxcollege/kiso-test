## 1. 複雑系 complex system

**局所的相互作用**により**大域的振る舞い**が生じるが、全体の振る舞いが個々の要素からは明らかでない系。

### 代表モデル

- セルオートマトン
- マルチエージェントシステム

### 特徴

- **べき乗則**、**フラクタル**（自己相似性）、**1/f ゆらぎ**
- **スケール不変性** — 特定の時空間スケールに依存しない
- その原理 → **自己組織化臨界（SOC）**

---

## 2. 自己組織化臨界 SOC ⭐

### 定義

系が**自分自身で臨界状態**に向かい、臨界状態を保とうとする性質。  
外部パラメータ（温度など）でチューニングする通常の臨界（相転移）とは対比。

### サンドパイルでの臨界

| 状態 | 意味 |
|------|------|
| 臨界状態 | 砂山が一定の高さ・傾きを保つ |
| 超臨界 | 次に崩壊する直前 |
| 亜臨界 | 臨界より低い |

砂粒を加え続けると、大小さまざまな**崩壊**が発生。

---

## 3. べき乗則 power law ⭐

観測確率（頻度）がサイズ \(x\) のべき乗に反比例:

\[
N(x) = C x^{-\tau}
\]

両対数プロットで**直線** → スケール不変。

### サンドパイル

**崩壊のサイズ**と**頻度**の関係がべき乗分布。  
大規模崩壊は稀だが、小規模と同じ法則で記述される（**黒いスワン**的リスクの背景）。

### パレート分布

連続型のべき乗則の代表。所得分布、地震、ネットワークの次数分布など。

---

## 4. カオス・フラクタル（用語）

| 概念 | 要点 |
|------|------|
| **カオス** | 初期値敏感（決定論的だが予測困難） |
| **フラクタル** | 部分が全体に似る（自己相似） |
| **1/f ノイズ** | 低周波成分が強いゆらぎ |

単純ルール → 複雑特性（**自己組織化**）の典型。

---

## 5. 複雑適応系（復習）

構成要素が環境に適応して振る舞いを変える。  
ミクロ・マクロ・ループで創発とフィードバック。

---

## 6. 講義全体のまとめ位置

```
マルコフ（確率）→ CA（局所ルール）→ エージェント（自律）
    → SOC（臨界・べき乗）→ 離散化（連続体の数値解法）
```
