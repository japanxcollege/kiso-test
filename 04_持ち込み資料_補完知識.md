## 確率の基礎（第 6-7 回）

📚 **原本**：[第06回 SE Basics 3](講義スライド一覧/第06回_確率モデル_System_Engineering_Basics_3.pdf) ・ [第07回 SE Basics 4](講義スライド一覧/第07回_確率分布_System_Engineering_Basics_4.pdf)

### 確率公理（Kolmogorov）

確率論の数学的基礎を、Kolmogorov が以下の 3 つの公理にまとめた：

1. **非負性**：任意の事象 A について **0 ≤ P(A) ≤ 1**
2. **全体の確率は 1**：**P(Ω) = 1**（Ω は標本空間）
3. **可算加法性**：互いに排反な事象 A₁, A₂, ... について

   $$P(A_1 \cup A_2 \cup \ldots) = P(A_1) + P(A_2) + \ldots$$

これだけから確率論のすべてが導かれる。

### 余事象・和の法則

- 余事象：**P(A^c) = 1 − P(A)**
- 和の法則：**P(A ∪ B) = P(A) + P(B) − P(A ∩ B)**
- 単調性：A ⊂ B ならば P(A) ≤ P(B)

### 条件付き確率とベイズの定理

**条件付き確率**：

$$P(A \mid B) = \frac{P(A \cap B)}{P(B)}, \quad P(B) > 0$$

**ベイズの定理**：

$$P(A \mid B) = \frac{P(B \mid A) P(A)}{P(B)}$$

応用：医療診断（症状から病気の確率）、スパムフィルタ、機械学習。

### 分布関数 F(x)

**累積分布関数（CDF）**：

$$F(x) = P(X \le x)$$

**3 つの性質**：

1. **単調非減少**：x₁ ≤ x₂ ならば F(x₁) ≤ F(x₂)
2. **右連続**：F(x+) = F(x)
3. **F(−∞) = 0, F(∞) = 1**

これは離散型・連続型のどちらの確率変数にも定義できる。

### 離散型と連続型の違い

| | 離散型 | 連続型 |
|---|--------|--------|
| 値 | 0, 1, 2, ... のような飛び飛び | 実数のあらゆる値 |
| 確率質量関数 PMF | p(x) = P(X = x) | — |
| 確率密度関数 PDF | — | f(x) ≥ 0、∫f = 1 |
| 期待値 | E[X] = Σ x_i p(x_i) | E[X] = ∫ x f(x) dx |
| P(X = x) | p(x)（正の値） | **0**（区間積分で扱う） |

注意：連続型では「ピッタリ x である確率」は 0。確率は**区間の積分**で与える。

### 代表分布 6 種の公式

| 分布 | パラメータ | 期待値 E[X] | 分散 V[X] | 用途 |
|------|-----------|-----------|-----------|------|
| ベルヌーイ | p | p | p(1−p) | 1 回の成功/失敗 |
| 二項 B(n, p) | n, p | np | np(1−p) | n 回独立試行の成功数 |
| ポアソン | λ | λ | λ | 単位時間の稀な事象数 |
| 幾何 | p | 1/p | (1−p)/p² | 初成功までの試行数 |
| 指数 | λ | 1/λ | 1/λ² | 待ち時間（無記憶性） |
| 正規 N(μ, σ²) | μ, σ² | μ | σ² | 測定誤差、自然現象 |

### 期待値の性質

**線形性**：E[aX + b] = aE[X] + b

**和の期待値**：E[X + Y] = E[X] + E[Y]（独立性不要）

**分散**：

$$V[X] = E[(X − E[X])^2] = E[X^2] − (E[X])^2$$

**独立な X, Y の分散**：V[X + Y] = V[X] + V[Y]

### 指数分布の無記憶性

$$P(X > s + t \mid X > s) = P(X > t)$$

意味：「これまで s 時間待ったという情報が、残り待ち時間の分布に影響しない」。マルコフ性と概念的に同型。M/M/1 待ち行列の理論的基盤。

### モンテカルロ法

複雑な期待値・確率を**乱数サンプル**で近似する方法。

- 乱数 X₁, X₂, ..., X_N をサンプル
- E[X] ≈ (1/N) Σ X_i
- **誤差は 1/√N のオーダー**（大数の法則）

応用：高次元積分、リスク評価、複雑系シミュレーション。

### 中心極限定理

独立同分布の確率変数 X_i の和は、n が大きくなると**正規分布に近づく**：

$$\frac{\bar{X} - \mu}{\sigma / \sqrt{n}} \to N(0, 1)$$

意味：個々の分布が何であれ、たくさん足せば正規分布になる。統計的推定・モンテカルロ誤差解析の基礎。

## マルコフ連鎖の詳細（第 8 回）

📚 **原本**：[第08回 SE Basics 5](講義スライド一覧/第08回_マルコフ待ち行列_System_Engineering_Basics_5.pdf)

### 推移確率行列の性質

- **p_{ij} ≥ 0**（非負）
- **Σ_j p_{ij} = 1**（各行の和 = 1）

### Chapman-Kolmogorov 方程式

$$p_{ij}^{(l+n)} = \sum_k p_{ik}^{(l)} p_{kj}^{(n)}$$

「l ステップで k に行き、その後 n ステップで j に行く確率の総和」。行列でいえば **P^(l+n) = P^l · P^n**。

### 状態の分類

| 用語 | 意味 |
|------|------|
| **既約**（irreducible） | 全状態が相互に到達可能 |
| **再帰的**（recurrent） | 有限時間で戻ってこれる |
| **過渡的**（transient） | 戻らない確率が正 |
| **非周期的**（aperiodic） | 周期が 1（戻り間隔がバラバラ） |
| **吸収状態** | p_ii = 1（そこから出られない） |
| **強マルコフ性** | 停止時刻以降、過去と独立 |

### 極限分布 = 定常分布の条件

次の 3 つが成り立つとき、極限分布が一意に存在し、定常分布と一致する：

1. **既約**（全状態が相互到達可能）
2. **非周期的**
3. **再帰的**（有限状態なら自動で成り立つ）

このとき、初期分布によらず：

$$\lim_{n \to \infty} p^{(0)} P^n = \pi$$

### M/M/1 の主要公式まとめ

| 量 | 公式 |
|----|------|
| 利用率 | ρ = λ/μ |
| 安定条件 | ρ < 1 |
| 系内平均人数 | L = ρ/(1−ρ) |
| 平均滞在時間 | W = 1/(μ−λ) |
| 平均待ち行列長 | L_q = ρ²/(1−ρ) |
| 平均待ち時間 | W_q = ρ/(μ−λ) |
| **リトルの法則** | **L = λW**（分布によらず） |

### ケンドール記法 A/B/c

- A：到着過程
- B：サービス過程
- c：サーバ数

例：
- M/M/1：ポアソン到着・指数サービス・1 サーバ
- M/M/c：c 台サーバ
- M/G/1：サービスが一般分布（Pollaczek-Khinchine 公式）

## オートマトン・CA（第 9 回）

📚 **原本**：[第09-10回 SE Basics 6](講義スライド一覧/第09-10回_オートマトンエージェント_System_Engineering_Basics_6.pdf)

### 有限オートマトンの形式定義

**5 つ組 (Q, Σ, δ, q₀, F)**：

- Q：**状態集合**
- Σ：**入力アルファベット**
- δ：**遷移関数** Q × Σ → Q（「状態 q で入力 a を受けたら、次の状態は何か」）
- q₀：**初期状態**
- F ⊆ Q：**受理状態**

用途：文法解析、字句解析、コンパイラ、正規表現マッチング。

### セルオートマトンの形式定義

**4 つ組 (L, S, N, f)**：

- L：**格子**（1 次元、2 次元など）
- S：**各セルの状態集合**
- N：**近傍関係**
- f：**更新関数**

CA の本質：**多数のセルが、同じルールで一斉に更新される**。

### 近傍の種類（2 次元の場合）

- **von Neumann 近傍**：**上下左右の 4 マス**
- **Moore 近傍**：**周囲 8 マス**（斜めも含む）

### Game of Life（Conway 1970）

各セルは「生」または「死」の 2 状態。Moore 近傍の生セル数で更新：

- **生**のセルは、隣接生セルが **2 か 3** で生存（それ以外は死亡）
- **死**のセルは、隣接生セルが **ちょうど 3** で誕生

このシンプルなルールから、

- 静物（変化しないパターン）
- 振動子（周期的に振動）
- グライダー（移動するパターン）
- グライダー銃（グライダーを永遠に生成）
- 任意の論理回路（**チューリング完全**）

までが創発する。

### Wolfram の CA 分類（1 次元）

| クラス | 挙動 | 例 |
|--------|------|-----|
| Class I | 均一に収束 | Rule 0 |
| Class II | 周期パターン | Rule 90 |
| Class III | カオス（ランダムに見える） | Rule 30 |
| Class IV | 複雑構造（最も興味深い） | Rule 110（チューリング完全） |

### サンドパイル詳細

ルール（既出）：

1. ランダムに 1 セルに +1
2. 値 ≥ 4 で 4 近傍に拡散（−4、隣接 +1）
3. 連鎖崩壊を 3 以下になるまで繰り返す

**結果**：

- 砂山は自然に**一定の傾き**を保つ
- 崩壊規模が**べき乗分布** N(x) ∝ x^(−τ)
- これが SOC の典型例

## エージェント・MAS（第 10 回）

📚 **原本**：[第09-10回 SE Basics 6](講義スライド一覧/第09-10回_オートマトンエージェント_System_Engineering_Basics_6.pdf) ・ [第10-11回 SE Basics 7](講義スライド一覧/第10-11回_エージェントSOC_System_Engineering_Basics_7.pdf)

### エージェントの 5 タイプ（Russell & Norvig）

| 番号 | タイプ | 特徴 | 例 |
|------|--------|------|-----|
| 1 | 単純反射 | 知覚 → 行動の IF-THEN ルール | サーモスタット |
| 2 | モデルベース反射 | 内部に世界モデル、見えない部分を推測 | 自動運転の地図 |
| 3 | 目標ベース | 目標達成のための手段選択 | 経路探索 |
| 4 | 効用ベース | 効用関数を最大化、合理的選択 | 投資ボット |
| 5 | 学習エージェント | 経験から自分を改善 | 強化学習 |

合理性のレベルが上がるにつれて、複雑な状況に対応できる。

### MAS（Multi-Agent System）

複数のエージェントが**相互作用**する系。

特徴：

- **集中制御がない**
- **局所相互作用**
- **創発**（個々の単純ルールから集団レベルで予測困難な性質が生まれる）

### 創発の代表例

| 系 | 局所ルール | 大域現象 |
|----|------------|----------|
| **Schelling 分居モデル**（1971） | 弱い人種選好 | **強い分居** |
| **Reynolds の boids** | 分離・整列・凝集 | 鳥群の飛行 |
| **アリのフェロモン** | 局所判断 | 最短経路発見 |
| 市場参加者 | 個別売買 | 価格動学・暴落 |
| 渋滞 | 個別追従行動 | 自然渋滞 |

### Schelling 分居モデルの教訓

「**個人レベルの差別意識が弱くても、社会レベルでは強い分離が出現する**」。創発の典型例で、**意図しない結果**の代表。

### Reynolds の boids（3 ルール）

1. **分離**：近くの個体から離れる
2. **整列**：周りの個体と方向を揃える
3. **凝集**：群れの中心に近づく

これだけで自然な群れ運動が生まれる。映画 CG でも使われている。

## 自己組織化臨界（第 11 回）

📚 **原本**：[第11回 SOC 原本](第11回_自己組織化臨界_原本.pdf) ・ [第10-11回 SE Basics 7](講義スライド一覧/第10-11回_エージェントSOC_System_Engineering_Basics_7.pdf)

### 複雑系の特徴

1. **べき乗則**：N(x) ∝ x^(−τ)
2. **フラクタル**（自己相似性）
3. **1/f ゆらぎ**
4. **スケール不変性**：どんなスケールで見ても同じ形状

### 自己組織化臨界（SOC）

**外部調整なしに、系が自分で臨界状態を保つ現象**。Bak, Tang, Wiesenfeld 1987。

通常の臨界（相転移）は外部パラメータ（温度など）でチューニングして到達する。SOC はパラメータ調整なしに自然に臨界に達する点が新規。

### べき乗則の現実例

| 現象 | 法則名 |
|------|--------|
| 地震のサイズ vs 頻度 | **Gutenberg-Richter 則** |
| 山火事の規模 | べき乗 |
| 金融市場の暴落 | べき乗 |
| 神経活動（ニューロン雪崩） | べき乗 |
| 種の絶滅 | べき乗 |
| 都市規模（人口 vs ランク） | **Zipf の法則** |
| 所得分布（高所得側） | **パレート分布** |

### パレート分布

連続型のべき乗則。確率密度関数は：

$$f(x) = \alpha x_m^{\alpha} / x^{\alpha + 1}, \quad x \ge x_m$$

x_m は下限、α は形状パラメータ。

### 「ブラック・スワン」とリスク管理

正規分布前提だと極端事象は「ほぼあり得ない」と評価される。**べき乗則の世界**では、

- 大規模イベントは稀だが、小規模と**同じ法則**で記述される
- **裾が太い**分布

つまり「100 年に 1 度の出来事」が**実は思ったより頻繁に起きる**。

正規分布前提のリスク評価では**過小評価**される（リーマンショック、福島原発事故）。これが「ブラック・スワン」問題。

### カオス・フラクタル・1/f ノイズ

| 概念 | 要点 | 代表例 |
|------|------|--------|
| **カオス** | 決定論的だが初期値敏感で予測困難 | ローレンツ方程式、ロジスティック写像、バタフライ効果 |
| **フラクタル** | 部分が全体に似る（自己相似） | マンデルブロ集合、海岸線、樹木の枝分かれ |
| **1/f ノイズ** | パワースペクトル ∝ 1/f | 心拍、音楽、太陽黒点 |

### ロジスティック写像

$$x_{n+1} = r x_n (1 − x_n)$$

r を変えると、収束 → 周期倍化 → カオス と挙動が変化する。1 次元カオスの代表例。

## 連続体の離散化（第 12 回）

📚 **原本**：[第12回 離散モデル化 原本](第12回_離散モデル化_原本.pdf)

### 離散化の 2 つの理由

1. **コンピュータは離散計算しかできない**：浮動小数点も有限精度
2. **連続体の PDE を解析的に解けない**：差分方程式に置き換えて数値解する

### 連続体近似

| 巨視変数 | 微視からの意味 |
|----------|----------------|
| 密度 ρ | 微小体積内の粒子数 / 体積 |
| 流速 v | 粒子の平均速度 |
| 内部エネルギー | 平均運動エネルギー |
| 圧力 p | 粒子の運動量交換 |

### Navier-Stokes 方程式（流体の運動方程式）

$$\frac{\partial \mathbf{v}}{\partial t} + (\mathbf{v}\cdot\nabla)\mathbf{v} = -\frac{1}{\rho}\nabla p + \frac{\mu}{\rho}\nabla^2 \mathbf{v} + \mathbf{F}$$

- **v**：流速
- **p**：圧力
- **ρ**：密度
- **μ**：粘性
- **F**：外力

実は数学的に「解が常に滑らかに存在するか」が**ミレニアム懸賞問題のひとつ（未解決）**。それでも近似的に数値解は計算できる。

### 離散化手法の比較

| 手法 | 概要 | 用途 |
|------|------|------|
| **有限差分法（FDM）** | 微分を**差分**で近似 | 単純な格子、CFD |
| **有限要素法（FEM）** | 要素分割 + **基底関数** | 構造解析、複雑形状 |
| **有限体積法（FVM）** | 体積積分の**保存則** | 流体（CFD）、保存則重視 |
| **粒子法（SPH 等）** | **ラグランジュ粒子**追跡 | 自由表面、衝突 |

### 数値解法の流れ

```
第一原理（分子動力学）
    ↓ 連続体近似
PDE（Navier-Stokes など）
    ↓ 離散化
代数方程式
    ↓ 計算機求解
数値解
```

### 格子と時間刻みのトレードオフ

- **空間**：細かい格子 → 精度↑、計算量↑
- **時間**：小さい Δt → 精度↑、計算量↑
- **CFL 条件**：双曲型 PDE では Δt は Δx より十分小さく取る必要がある（数値発散の防止）

### メゾスコピックモデル

ミクロとマクロの**中間スケール**。例：
- **格子ボルツマン法**（流体）
- **粗視化シミュレーション**（高分子）

## ファジィ理論の詳細（第 4 回）

📚 **原本**：[第05回 ファジィ](第05回_ファジィ_原本/第05回_ファジィ_スライド.pdf) ・ [演習ツール](第05回_ファジィ_原本/fuzzy_exercise_tool_3.html)

### メンバーシップ関数

要素 x の集合 A への所属度を [0, 1] で表す：

$$\mu_A(x) \in [0, 1]$$

- μ = 1：完全に所属
- μ = 0：完全に非所属
- 中間値：部分的所属

例：「若い」のメンバーシップ：

- μ(20歳) = 1.0
- μ(25歳) = 0.9
- μ(40歳) = 0.5
- μ(60歳) = 0.1

### ファジィ演算

- **AND（t-norm）**：典型 **min(μ_A, μ_B)**、積 μ_A · μ_B も可
- **OR（s-norm）**：典型 **max(μ_A, μ_B)**、確率和 μ_A + μ_B − μ_A μ_B も可
- **NOT**：**1 − μ_A**

### ファジィ推論の流れ

1. **ファジィ化**：明確な入力値（例：温度 28℃）→ メンバーシップ度（「やや暑い」0.7）
2. **ルール適用**：「もし暑いなら冷房を強くせよ」などのルール群
3. **合成**：複数ルールの結果を集約
4. **デファジィ化**：ファジィ結果 → 明確な出力値（重心法など）

応用：エアコン自動制御、洗濯機、新幹線の自動運転制御。

### FCM（Fuzzy Cognitive Map）

Kosko（1986）。Cognitive Map（符号付き因果）に**ファジィな重み**（[−1, +1] の値）を持たせたもの。

- Cognitive Map：**符号のみ**（+/−）
- DEMATEL：**強さのみ**（0〜4、非負）
- FCM：**符号と強さ両方**（[−1, +1]）

ファジィ行列演算 + 反復計算で系の挙動を予測する。

## AHP の詳細（第 5 回）

📚 **原本**：[第02-03回 SE Basics 2](講義スライド一覧/第02-03回_System_Engineering_Basics_2.pdf)

### 階層構造

3 層：

1. **目標**（最上位）：例「最適な新車を選ぶ」
2. **評価基準**（中間）：価格、燃費、安全性、デザイン
3. **代替案**（最下位）：車 A、車 B、車 C

### Saaty の 1〜9 尺度

| 値 | 意味 |
|----|------|
| 1 | 同等 |
| 3 | やや優 |
| 5 | 優 |
| 7 | かなり優 |
| 9 | 極めて優 |
| 2, 4, 6, 8 | 中間値 |
| 1/3, 1/5, ... | 逆の関係 |

### 一対比較行列の性質

- **正逆対称**：a_{ji} = 1/a_{ij}
- **対角要素 = 1**：a_{ii} = 1
- **3×3 なら 3 つの独立な比較**（n(n−1)/2）

### 重みの算出（2 通り）

**1. 固有ベクトル法**：比較行列の最大固有値 λ_max に対応する固有ベクトルを正規化

**2. 幾何平均法（簡略法）**：

- 各行の幾何平均を計算
- 合計で割って正規化

### 整合性指標

**CI（Consistency Index）**：

$$\mathrm{CI} = \frac{\lambda_{\max} - n}{n - 1}$$

**CR（Consistency Ratio）**：

$$\mathrm{CR} = \frac{\mathrm{CI}}{\mathrm{RI}}$$

RI（Random Index）は n に依存：

| n | 2 | 3 | 4 | 5 | 6 | 7 |
|---|---|---|---|---|---|---|
| RI | 0 | 0.58 | 0.90 | 1.12 | 1.24 | 1.32 |

**CR < 0.1** なら一貫した判断。

### AHP の注意点

- 一対比較数 = n(n−1)/2 → 要素 10 個で 45 回 → **負担大**
- **ランキング逆転問題**：代替追加で順位が変わる
- 主観的バイアス → CR でチェック

### AHP と他手法の使い分け

| 手法 | 問い | 出力 |
|------|------|------|
| ISM | 要素間の構造・階層は？ | 階層図 |
| DEMATEL | 影響の強さ・原因/結果は？ | D−R グラフ |
| Cognitive Map | 因果の正負は？ | 符号付きグラフ |
| **AHP** | **どの案を選ぶか？** | **優先順位** |
| SD | 時間とともにどう変わるか？ | 時系列 |

### 悪定義問題（wicked problem）

Rittel & Webber（1973）が提唱。次のような特徴を持つ：

- 何が「正しい解」か関係者で意見が分かれる
- 解は「正しい/間違い」ではなく「より良い/悪い」で評価
- 問題自体が**やってみないと分からない**
- 介入が新たな問題を生む

例：気候変動対策、貧困問題、まちづくり、教育改革。

- **C**ustomer：受益者
- **A**ctor：実行者
- **T**ransformation：変換（インプット → アウトプット）
- **W**eltanschauung：世界観
- **O**wner：オーナー
- **E**nvironmental constraints：環境制約

## ISM の補完知識（第 2 回）

📚 **原本**：[第02回スライド](講義スライド一覧/第02回_スライド.pdf) ・ [構造化レポート](第04回_構造化レポート演習/構造化レポート.pdf)

### ブロック三角化

強連結成分（互いに到達可能なノード群）をまとめ、階層順に並べ替える前処理。これで全体構造が見やすくなる。

### 骨格行列（skeleton matrix）

A → B → C のとき、推移律で派生する A → C のような**冗長な関係**を除いた**最小構造関係**。

可到達行列から推移閉包を取った後、冗長関係を除いて骨格行列を得る。これが ISM 階層図の元になる。

### 推移律の反例

- **友人関係**：A と B が友人、B と C が友人でも、A と C は友人とは限らない
- **じゃんけん**：グー > チョキ > パー > グー（循環）
- **相性**：A と B が相性◎、B と C が相性◎でも、A と C は相性 × かも

→ こういう関係は ISM では扱えない。DEMATEL や Cognitive Map を使う。

## DEMATEL の補完知識（第 2 回）

📚 **原本**：[第02回スライド](講義スライド一覧/第02回_スライド.pdf) ・ [構造化レポート](第04回_構造化レポート演習/構造化レポート.pdf) ・ [Excel 計算ツール](第04回_構造化レポート演習/ISM-DEMATEL計算ツール_A.xlsm)

### なぜ正規化するか

T = N + N² + N³ + ... という級数が**収束する**ことを保証するため。N の固有値（スペクトル半径）が 1 未満であれば、級数は収束する。**最大行和で割ることでスペクトル半径 < 1 を保証**できる。

### N の冪零性

N が**上三角で対角が 0** の場合、n × n 行列なら N^n = 0（冪零）。

このとき：

$$(I - N)^{-1} = I + N + N^2 + \ldots + N^{n-1}$$

の有限和で計算できる（無限和を計算しなくて済む）。

### D−R と D+R の解釈

| 値 | 解釈 |
|----|------|
| **D−R > 0、D+R 大** | 強い原因系。介入すべき根本要素 |
| **D−R > 0、D+R 小** | 自律的な原因系。影響範囲は限定的 |
| **D−R < 0、D+R 大** | 中心的な結果系。対症療法の対象 |
| **D−R < 0、D+R 小** | 末端の結果系。直接介入は無意味 |

これを D−R を縦軸、D+R を横軸にプロットしたのが **NRM（Network Relation Map）**。
